# 世界观设定

## 诡异复苏背景

三年前，全球发生了一场神秘事件——"诡异复苏"。

无数传说中的鬼怪、妖魔从暗处现身，开始侵蚀人类的世界。

各国政府秘密成立了"超凡局"，负责处理诡异事件，保护普通民众。

## 诡异分类

### 按等级分类

| 等级 | 危害范围 | 代表 | 对应字数出现 |
|------|----------|------|-------------|
| E级 | 个人 | 普通厉鬼、低级妖怪 | 0-10万 |
| D级 | 家庭 | 怨灵、小妖 | 5-15万 |
| C级 | 社区 | 凶灵、中级妖怪 | 10-20万 |
| B级 | 街区 | 厉鬼王、大妖 | 15-25万 |
| A级 | 城市 | 鬼将、妖王 | 20-30万 |
| S级 | 省级 | 鬼王、大妖王 | 30-80万 |
| SS级 | 国家 | 鬼帝、妖帝 | 80-150万 |
| SSS级 | 大陆 | 鬼神、妖神 | 150-220万 |
| 神级 | 全球 | 诡神眷属 | 220-260万 |
| 神王级 | 星系 | 诡神 | 260-280万 |
| 神帝级 | 宇宙 | 诡神之祖 | 280-300万 |

### 按类型分类

1. **鬼类**：人死后的怨念凝聚
2. **妖类**：动植物修炼成精
3. **怪类**：异常现象的具象化
4. **邪类**：外来入侵的存在
5. **诡神**：超越理解的高维存在（200万字后才出现）

4. **黑巫教**：崇拜诡神的邪恶组织（主要反派）

## 地理设定

### 第一卷地图：江城

- **位置**：华中地区三线城市
- **人口**：500万
- **超凡局**：市级分局，约200名驱鬼师
- **特色**：
  - 万人坑（主角觉醒地）
  - 老城区（诡异高发区）
  - 地下遗迹（远古秘境，伏笔）
  
### 重要地点

1. **万人坑**：抗战时期的乱葬岗，诡气最重
2. **鬼市**：每月十五出现的诡异集市
3. **幽冥路**：连接阴阳两界的通道
4. **王家大宅**：江城最大世家（反派）
