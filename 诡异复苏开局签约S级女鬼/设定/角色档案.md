# 角色档案 - 《诡异复苏：开局签约S级女鬼》

---

## 一、主角

### 陈渊

**基础信息**
- 年龄：21岁
- 身份：江城大学大三学生，计算机专业
- 外貌：身高178，长相普通偏上，但有一双很深邃的眼睛
- 性格：表面随和，内心冷静，被逼到绝境时极度果断

**前期状态**
- 孤儿出身，靠奖学金读书
- 被室友赵凯陷害，推入S级鬼域"万人坑"
- 所有人都以为他必死

**金手指：鬼契系统**
- 核心能力：与诡异签订契约，收服为己用
- 契约条件：每个诡异有不同的签约条件
- 成长机制：
  - 契约诡异越强，自身实力越强
  - 可以借用契约诡异的能力
  - 诡异也会随着陈渊变强而进化
- 限制：
  - 同时激活的契约数量有限（初期3个）
  - 契约诡异有自主意识，不是工具人
  - 某些诡异有契约代价

**性格标签**
- 表面：人畜无害、好说话
- 内核：冷静、果断、恩怨分明
- 口头禅："有意思。"
- 处事原则：不主动惹事，但谁惹我，我灭谁全家

**角色弧光（300万字）**
| 阶段 | 字数 | 状态变化 |
|------|------|----------|
| 开局 | 0-30万 | 从被陷害的普通人 → 崛起的新星驱鬼师 |
| 崛起 | 30-80万 | 从野路子 → 被国家超凡局招揽 |
| 中期 | 80-150万 | 从执行者 → 发现世界真相 |
| 后期 | 150-250万 | 从棋子 → 掀翻棋盘的人 |
| 终局 | 250-300万 | 从人类 → 人与诡异共存的开创者 |

## 二、女主 / 契约诡异

### 厉姬（第一女主 / S级诡异）

**基础信息**
- 诡异等级：S级
- 别称：千年厉姬、万人坑之主
- 外貌：红衣黑发，肤白如雪，美艳至极，眼角一颗泪痣
- 形态：平时可化为人形，战斗时显露鬼态

**生前身份**
- 千年前的将门之女，名为"姬红妆"
- 被奸臣陷害，与三千将士一同被活埋于万人坑
- 怨念不散，化为S级厉鬼

**性格**
- 高冷、傲娇、护短
- 对陈渊：从利用 → 好奇 → 认可 → 暧昧
- 口头禅："本座……" "你这凡人……"
- 特点：嘴上嫌弃，身体诚实

**契约条件**
- 帮她找到千年前陷害她的奸臣后裔
- 帮她的三千怨魂超度（长期任务）

**能力**
- 厉鬼之威：精神压制，弱者直接吓死
- 血衣护主：化为血色护盾保护契约者
- 万鬼朝宗：召唤万人坑中的鬼兵（大招）

**感情线节奏**
| 阶段 | 关系 |
|------|------|
| 初识 | 契约关系，互相利用 |
| 30万字 | 产生好奇，开始关注 |
| 80万字 | 认可实力，愿意并肩作战 |
| 150万字 | 暧昧萌芽，醋意初现 |
| 250万字 | 感情明确，人鬼情深 |

### 白九（第二女主 / A级诡异）

**基础信息**
- 诡异等级：A级（有进化潜力）
- 类型：狐妖
- 外貌：银发赤瞳，身材娇小，看起来只有十五六岁
- 来源：校园七大不可思议之"图书馆狐仙"

**性格**
- 活泼、黏人、吃货属性
- 与厉姬形成反差（一冷一热）
- 口头禅："主人~" "饿了~"

**契约条件**
- 每天要吃一份炸鸡（低门槛）
- 但有隐藏条件……

**能力**
- 幻术：迷惑敌人心智
- 灵狐之速：极速移动
- 九尾觉醒：每觉醒一尾，实力翻倍（成长型）

### 沈清音（人类女主）

**基础信息**
- 年龄：23岁
- 身份：国家超凡局特别行动组成员
- 外貌：英姿飒爽，马尾，制服控福利
- 家世：驱鬼世家，从小接受训练

**性格**
- 高冷、认真、有点死板
- 对规则的执着与陈渊的野路子形成冲突
- 傲娇属性，嘴硬心软

**与主角关系**
- 初期：调查陈渊（怎么可能签约S级诡异？）
- 中期：合作任务，产生信任
- 后期：并肩作战，感情线展开

## 三、反派体系

### 第一卷BOSS：赵凯（室友）
- 身份：陷害主角的室友
- 动机：嫉妒 + 被人利用
- 结局：被厉姬吓成废人（第3章）

### 第一卷幕后：王家
- 江城驱鬼世家
- 想要万人坑中的宝物，利用赵凯试探
- 与主角结仇

### 第二卷BOSS：王天霸
- 王家嫡系天才
- B级驱鬼师，看不起野路子
- 多次针对主角，最终被打脸

- 邪恶组织，信奉"诡神"
- 想要制造更多诡异祸乱人间
- 与主角多次交手
