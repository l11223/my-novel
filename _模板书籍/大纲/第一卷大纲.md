# 第一卷大纲：起步期

> ⚠️ **硬性要求**：0-30万字 | 第3境界上限 | 不能离开第一张地图

---

## 本卷基本信息

| 项目 | 要求 |
|------|------|
| **字数范围** | 0 - 30万字 |
| **境界范围** | 第1境界 → 第3境界 |
| **地图范围** | 只能在第一张地图 |
| **敌人上限** | 第3境界 |
| **主线进度** | 0% → 10% |

---

## 🔴 本卷禁止事项

- ❌ 主角境界超过第3境界
- ❌ 离开第一张地图
- ❌ 出现第4境界以上的敌人
- ❌ 终极boss出场（连提都不能提）
- ❌ 揭示世界真相
- ❌ 主线推进超过10%
- ❌ 任何形式的收尾或结局

---

## 本卷剧情规划

### 第一阶段（0-10万字）：觉醒

**字数**：0-10万
**境界**：第1境界 → 第2境界
**关键词**：觉醒、初战、立足

#### 剧情点
1. [开局冲突] - 第1章
2. [获得金手指] - 第1-2章
3. [第一次战斗] - 第3-5章
4. [第一次打脸] - 第5-8章
5. [加入势力/建立据点] - 第8-15章
6. [突破第2境界] - 第15-20章

#### 本阶段必须完成
- [ ] 主角获得金手指
- [ ] 第一个敌人出现（但不彻底解决）
- [ ] 建立至少一个盟友关系
- [ ] 埋下至少3个伏笔

---

### 第二阶段（10-20万字）：站稳

**字数**：10-20万
**境界**：第2境界稳固
**关键词**：扩张、冲突、名气

#### 剧情点
1. [势力扩张]
2. [与主要反派第一次正面冲突]
3. [获得重要资源/传承]
4. [打出名气]
5. [新的盟友加入]

#### 本阶段必须完成
- [ ] 与本卷主要反派正式交锋
- [ ] 主角在本地打出名气
- [ ] 至少2个新配角登场
- [ ] 感情线开始萌芽

---

### 第三阶段（20-30万字）：初露锋芒

**字数**：20-30万
**境界**：第2境界巅峰 → 第3境界
**关键词**：突破、大战、留悬念

#### 剧情点
1. [与反派的大战]
2. [突破第3境界]
3. [反派初步败退（但没有彻底解决）]
4. [更大势力的影子出现（伏笔）]
5. [离开第一张地图的契机出现]

#### 本阶段必须完成
- [ ] 主角突破第3境界
- [ ] 本卷主要反派受挫（但逃跑/有后台）
- [ ] 揭示更大的世界存在
- [ ] 为第二卷做铺垫

---

## 本卷结尾要求

```
✅ 必须做到：
- 主角在本地站稳脚跟
- 有前往下一张地图的理由
- 留下至少3个未解之谜
- 至少1个敌人没有被彻底解决
- 感情线只是刚开始

❌ 绝对不能：
- 写成结局的样子
- 让主角称霸本地
- 解决所有问题
- 揭示太多真相
```
