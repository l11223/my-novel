# 📚 我的小说仓库

用 AI 写 300 万字长篇网文的完整工作流。

---

## 📁 目录结构

```
my-novel/
├── WRITING_PROMPT.md      # 核心提示词（发给AI用）
├── QUICK_START.md         # 快速开始指南
├── _模板书籍/              # 新书模板（复制这个开始新书）
│   ├── 章节/
│   ├── 设定/
│   ├── 大纲/
│   ├── 草稿/
│   └── 存档/
├── 我的第一本书/           # ← 你的书（复制模板后改名）
├── 我的第二本书/           # ← 另一本书
└── ...
```

---

## 🚀 如何开始写新书

### 第一步：复制模板
```bash
cp -r _模板书籍 "你的书名"
```

### 第二步：填写设定
打开 `你的书名/设定/` 和 `你的书名/大纲/`，填写角色和世界观。

### 第三步：发送提示词
1. 复制 `WRITING_PROMPT.md` 的**全部内容**
2. 粘贴到 AI 对话框
3. 回答 AI 的题材选择问题
4. 开始创作！

### 第四步：保存章节
把 AI 输出的章节内容保存到 `你的书名/章节/` 目录。

### 第五步：定期存档
每写完一卷，让 AI 生成存档，保存到 `你的书名/存档/` 目录。

---

## 📝 日常写作流程

```
1. 打开 AI 对话
2. 粘贴提示词（或读取存档）
3. 发送 "继续" 让 AI 写下一章
4. 把章节内容复制到仓库
5. git add . && git commit -m "更新到第xxx章" && git push
```

---

## 💡 常用指令

| 指令 | 作用 |
|------|------|
| `继续` | 写下一章 |
| `存档` | 生成当前进度存档 |
| `检查进度` | 显示字数和完成百分比 |
| `加爽点` | 下一章安排装逼打脸 |
| `扩展世界观` | 引入新地图/新势力 |
