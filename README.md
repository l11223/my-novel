# 📚 我的小说项目

这是一个用于创作和管理小说的仓库。

## 📁 目录结构

```
.
├── chapters/       # 正式章节
├── characters/     # 角色设定
├── worldbuilding/  # 世界观设定
├── outlines/       # 故事大纲
├── drafts/         # 草稿区
└── notes/          # 灵感笔记
```

## 🚀 快速开始

1. 在 `outlines/` 中规划你的故事大纲
2. 在 `characters/` 中创建角色档案
3. 在 `worldbuilding/` 中构建世界观
4. 在 `drafts/` 中自由写作草稿
5. 将完成的章节整理到 `chapters/` 中

## 📝 写作建议

- 每个章节建议单独一个 `.md` 文件
- 使用 `001-章节名.md` 格式命名以保持顺序
- 随时在 `notes/` 中记录灵感
- 定期 commit 保存你的进度
