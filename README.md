# 📚 AI 网文写作系统 v10.0

> 🏆 集合 15 位白金大神精华 | 3200+ 行提示词 | 300万字长篇小说专用

## ⚡ 快速开始（3步）

### 1️⃣ 复制提示词
打开 `WRITING_PROMPT.md`，全选复制（Cmd+A, Cmd+C）

### 2️⃣ 粘贴给 AI  
打开 Claude/ChatGPT/通义千问，粘贴发送

### 3️⃣ 回答问题，开始写
AI 问你题材、主角、风格，回答后就开始输出小说
