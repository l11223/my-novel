# 长篇网文创作系统

> 300万字级别长篇小说创作与管理系统

---

## 快速开始

1. 阅读 `WRITING_PROMPT.md` 了解创作核心原则
2. 使用启动指令确定题材、主角、风格
3. 在 `outlines/` 中规划故事大纲
4. 在 `characters/` 中建立角色档案
5. 在 `worldbuilding/` 中构建世界观
6. 在 `chapters/` 中开始正式创作

## 创作原则速查

### 黄金三章法则
- 第一章：5秒抓住读者
- 前三章完成：出场 → 金手指 → 第一次打脸

### 爽点密度
| 类型 | 间隔 | 示例 |
|------|------|------|
| 小爽点 | 2000字 | 打脸/突破/获宝 |
| 中爽点 | 5000字 | 阶段胜利/敌人求饶 |
| 大爽点 | 2万字 | 越级杀敌/扬名 |
| 高潮 | 10万字 | BOSS战/大型打脸 |

### 结构框架（300万字）

| 卷 | 字数 | 核心内容 |
|-----|------|----------|
| 开篇卷 | 30万 | 起源地图，建立根基 |
| 崛起卷 | 50万 | 扩大势力，名声渐起 |
| 争霸卷 | 60万 | 多方角逐，异军突起 |
| 称雄卷 | 50万 | 称霸一方，新世界 |
| 飞升卷 | 40万 | 高级地图，重新开始 |
| 巅峰卷 | 40万 | 顶级对决，揭示背景 |
| 终极卷 | 30万 | 最终决战，完美收尾 |

## 工作流程

### 阶段一：企划
1. 确定题材和风格
2. 设计金手指和成长体系
3. 完成总纲和分卷大纲
4. 建立核心角色档案

### 阶段二：创作
1. 按章节输出正文（每章2000-3000字）
2. 每章结尾设置钩子
3. 按规定密度安排爽点
4. 埋设伏笔并记录

### 阶段三：复盘
1. 每10章检查节奏和人设一致性
2. 每卷结束总结并规划下卷
3. 定期检查伏笔回收情况

## 启动创作

在开始之前，请确定：

1. **题材**：玄幻/仙侠/都市/科幻/历史/游戏
2. **主角**：性别、性格、金手指类型
3. **风格**：纯爽文/有深度/轻松搞笑/热血燃向
4. **感情线**：后宫/单女主/无CP

确认后，开始填充 `outlines/总纲.md` 和创建角色档案。
