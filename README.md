## 📊 300万字规划

| 阶段 | 字数 | 主角境界上限 | 禁止事项 |
|------|------|-------------|----------|
| 起步期 | 0-30万 | 第3境界 | 不能离开第一张地图 |
| 成长期 | 30-80万 | 第5境界 | 终极boss不能出场 |
| 崛起期 | 80-150万 | 第7境界 | 不能揭示全部真相 |
| 称霸期 | 150-220万 | 第8境界 | 不能写大结局 |
| 巅峰期 | 220-280万 | 第9境界 | 不能让所有敌人死光 |
| 终局期 | 280-300万 | 巅峰 | **只有这里可以写结局** |

## 🚀 快速开始

### 1. 创建新书

```bash
./new_book.sh "你的书名"
```

### 2. 填写设定

- `你的书名/设定/世界观.md` - 世界背景
- `你的书名/设定/角色档案.md` - 角色信息
- `你的书名/大纲/总大纲.md` - 剧情规划

### 3. 开始写作

1. 把 `WRITING_PROMPT.md` 发给AI
2. **每次写作前让AI先读取 `进度锁.md`**
3. 写完每章后更新进度锁

---

## ⚠️ 防止提前结局的机制

### 进度锁系统

每次写作前，AI必须：
1. 读取 `进度锁.md`
2. 确认当前字数和阶段
3. 确认本阶段的禁止事项
4. 写完后更新进度锁

### 阶段锁定

- 字数不到，不能升级
- 字数不到，不能换地图
- 字数不到，不能打boss
- 字数不到，不能写结局
