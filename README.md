## ⚡ 快速开始（3步）

### 1️⃣ 复制提示词
打开 `WRITING_PROMPT.md`，全选复制（Cmd+A, Cmd+C）

### 2️⃣ 粘贴给 AI  
打开 Claude/ChatGPT/通义千问，粘贴发送

### 3️⃣ 回答问题，开始写
AI 问你题材、主角、风格，回答后就开始输出小说

## ⚠️ 注意事项

1. 提示词很长，部分AI可能无法完整处理，可用 `QUICK_START.md`
2. 建议分卷创作，利用存档系统续写
3. AI有上下文限制，超长对话需要开新对话读档继续
4. 最终作品需要人工校对和润色
