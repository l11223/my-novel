# 《诡异觉醒：开局签约SSS级鬼王》世界观设定

---

## 一、基础设定

### 世界类型
- [x] 都市异能（诡异复苏、灵异契约）
- [x] 现代背景（202X年）
- [x] 隐藏世界（诡异空间、鬼域）

### 核心设定
- **时代背景**：202X年，全球诡异复苏元年后的第三年
- **世界名称**：蓝星（现实世界）+ 诡异空间（叠加位面）
- **核心冲突**：人类vs诡异、各国超凡势力博弈、诡神降临危机
- **独特法则**：灵魂契约、诡异规则、因果律
- **世界秘密**：诡异复苏的真正原因——诡神即将降临，而这一切都是某个存在的布局

### 诡异复苏背景
三年前，全球同时爆发诡异事件：
- 各种都市传说成真
- 鬼怪从阴间渗透到阳间
- 少数人觉醒灵异能力
- 各国成立超凡机构应对

## 六、能力体系

### 觉醒者能力分类
| 类型 | 特点 | 代表能力 |
|------|------|----------|
| 战斗型 | 直接战力 | 灵力强化、元素操控 |
| 感知型 | 探测能力 | 灵视、预知、追踪 |
| 辅助型 | 支援能力 | 治愈、结界、加持 |
| 特殊型 | 罕见能力 | 契约、时间、空间 |

### 主角能力：灵魂契约
**核心机制**：
- 可与诡异生物签订灵魂契约
- 契约后可共享对方能力
- 契约越深，共享程度越高
- 契约数量随等级提升

**限制**：
- 需要双方自愿（或一方濒死）
- 契约破裂会反噬
- 契约对象等级不能超过自身太多

**成长路线**：
- 初期：1个契约槽，共享10%
- 中期：3个契约槽，共享30%
- 后期：7个契约槽，共享70%
- 巅峰：无限契约，完全共享

## 七、诡异规则

### 诡异特性
1. **忌讳**：每个诡异都有特定忌讳，触犯必死
2. **规则**：诡异遵循特定规则行动
3. **弱点**：找到规则就能找到破解方法
4. **领域**：高级诡异有自己的鬼域

### 常见诡异类型
| 类型 | 来源 | 特点 | 对抗方式 |
|------|------|------|----------|
| 怨念型 | 人死后的执念 | 执念不灭，鬼不散 | 化解执念 |
| 诅咒型 | 恶意诅咒凝聚 | 会传播扩散 | 破解诅咒源 |
| 自然型 | 天地怨气 | 无目标攻击 | 净化 |
| 古老型 | 远古存在 | 强大且智慧 | 封印或契约 |
| 诡神眷属 | 诡神创造 | 近乎不死 | 神级力量 |

- 诡神曾被封印

- 第一代契约者就是封印诡神的人
- 主角是被选中对抗诡神的人
- 最终Boss是第一个背叛的契约者

## 十、特殊设定

### 灵魂契约详细规则
1. **契约形成**：需要双方灵魂接触
2. **契约层次**：
   - 初级契约：简单联系，可单方面解除
   - 中级契约：能力共享，解除需双方同意
   - 高级契约：生死与共，一方死亡另一方重伤
   - 至高契约：灵魂融合，永不分离

3. **契约好处**：
   - 主人可使用契约者部分能力
   - 契约者获得主人的生命力补充
   - 双方可心灵沟通

4. **契约限制**：
   - 等级差距过大无法契约
   - 契约数量有上限
   - 违背契约会受到反噬

### 鬼域设定
- **定义**：诡异创造的独立空间
- **特点**：内部规则由诡异制定
- **进入方式**：特定触发条件
- **逃脱方式**：破解规则或击杀核心诡异
