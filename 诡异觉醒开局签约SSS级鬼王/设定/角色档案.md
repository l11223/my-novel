# 《诡异觉醒：开局签约SSS级鬼王》角色档案

---

## 一、主角档案

### 基础信息
- **姓名**：林渊
- **年龄**：开局22岁，结局约30岁（因契约延寿，外表不变）
- **身份**：普通大学毕业生 → 契约者 → 人类最强 → 诡神终结者
- **外貌关键词**：黑发、眼神深邃、嘴角常带若有若无的笑

### 性格设定
- **主性格标签**：腹黑（表面人畜无害，内心算计深远）
- **副性格标签**：护短（动他的人一个都别想好过）
- **行为准则**：恩怨分明，有仇必报，有恩必还
- **底线/逆鳞**：伤害他契约的伙伴
- **性格弱点**：有时过于相信契约伙伴

### 说话风格
- **常用句式**：
  - "有意思。"
  - "是吗？"
  - "那就试试看。"
- **口头禅**："记住今天。"
- **绝不会说的话**："求你了""我认输"
- **标志性动作**：嘴角微扬、双手插兜

### 金手指设定：灵魂契约系统
- **类型**：契约型系统
- **核心能力**：与诡异生物签订灵魂契约，共享能力
- **限制/代价**：
  - 契约数量有上限
  - 等级差距过大无法契约
  - 契约需要双方同意
- **成长路线**：
  - 10万字解锁：中级契约（3个槽位，共享30%）
  - 50万字解锁：高级契约（5个槽位，共享50%）
  - 100万字解锁：王级契约（7个槽位，共享70%）
  - 200万字解锁：帝级契约（9个槽位，共享80%）
  - 最终形态：神级契约（无限槽位，完全共享）

### 角色弧光规划
```
【天真期】0-30万
- 状态：刚觉醒，对诡异世界充满好奇和恐惧
- 关键成长：学会在这个世界生存

【受挫期】30-80万
- 状态：遭遇重大打击（契约伙伴受伤/背叛）
- 关键成长：理解契约的真正含义

【觉醒期】80-150万
- 状态：确立自己的"道"——守护所有契约之人
- 关键成长：从被动应对到主动出击

【巅峰期】150-250万
- 状态：成为人类顶尖战力
- 关键成长：承担拯救世界的责任

【超越期】250-300万
- 状态：继承第一代契约者的意志
- 关键成长：完成最终使命，封印诡神
```

## 二、契约伙伴档案

### 契约者1：苏幽（女主·SSS级鬼王）
- **真名**：苏幽冥
- **类型**：高冷女鬼、病娇（对主角）
- **身份**：远古SSS级鬼王，被封印万年
- **外貌记忆点**：
  - 一袭白衣，长发及腰
  - 肤白如雪，美若天仙
  - 眼眸深处有幽蓝色光芒
- **性格关键词**：高冷、强大、占有欲强
- **说话风格**：
  - "本座不屑于此。"
  - "你是本座的人。"
- **与主角关系**：契约伙伴 → 灵魂伴侣
- **能力**：
  - 死亡凝视（A级以下直接击杀）
  - 鬼域召唤（创造独立空间）
  - 灵魂操控（控制亡灵大军）
  - 虚无之力（本源攻击）
- **封印原因**：万年前为封印诡神而自我封印
- **解封条件**：遇到真正的契约者

**感情线节点**：
| 节点 | 字数 | 事件 | 好感度变化 |
|------|------|------|------------|
| 初遇 | 第1章 | 被主角意外唤醒并契约 | 0 → 10（不屑但有兴趣） |
| 认可 | 10万字 | 主角为救她受重伤 | 20 → 50（刮目相看） |
| 心动 | 30万字 | 并肩作战，灵魂共鸣 | 50 → 70（产生感情） |
| 确认 | 60万字 | 互相表白 | 70 → 90 |
| 波折 | 100万字 | 苏幽记忆恢复，得知主角可能是她仇人后代 | 90 → 70 |
| 圆满 | 150万字 | 误会解开，升级为至高契约 | 100 |

### 契约者2：白小七（萝莉鬼差·A级）
- **身份**：地府鬼差，因地府崩塌流落人间
- **外貌**：12岁左右的小女孩，双马尾，穿黑色连衣裙
- **性格**：活泼、贪吃、忠诚
- **口头禅**："七七要吃好吃的！"
- **能力**：
  - 追魂索命（追踪任何目标）
  - 鬼差令（短暂召唤亡魂）
  - 虚空行走（短距离传送）
- **契约时间**：约第20万字
- **与主角关系**：妹妹一般的存在

### 契约者3：骨帝（骷髅王·S级）
- **身份**：远古战场亡魂汇聚而成的骷髅王
- **外貌**：三米高的骨骼巨人，眼眶中燃烧蓝色火焰
- **性格**：沉默、好战、忠诚
- **能力**：
  - 不死军团（召唤骷髅大军）
  - 骨化领域（范围弱化敌人）
  - 死亡冲锋（极致攻击）
- **契约时间**：约第50万字
- **与主角关系**：最可靠的战力

### 契约者4：影鬼（暗影型·B级→S级）
- **身份**：主角影子中诞生的诡异
- **外貌**：纯黑色的人形阴影
- **性格**：无口、执行力强
- **能力**：
  - 暗影潜行（隐身）
  - 影杀（暗杀）
  - 影分身（分身）
- **契约时间**：约第30万字
- **与主角关系**：最忠诚的暗卫

### 契约者5：九尾妖狐·小九（妖族·A级）
- **身份**：上古妖族后裔
- **外貌**：化形为妩媚女子，九条白色狐尾
- **性格**：妩媚、聪明、有时腹黑
- **能力**：
  - 魅惑（精神攻击）
  - 狐火（蓝色火焰）
  - 幻术（制造幻境）
- **契约时间**：约第80万字
- **与主角关系**：亦敌亦友后成为契约者
