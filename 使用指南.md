# 📖 使用指南（必读）

## ⚠️ 重要：AI 读不了链接！

发 GitHub 链接给 AI，它看不到内容。
必须**复制文字**发过去。

---

## 🚀 三步开始写小说

### 第一步：复制提示词

打开 `WRITING_PROMPT.md` 文件，**全选复制**（Cmd+A, Cmd+C）

> 提示词有 3200+ 行，如果 AI 说太长，用 `QUICK_START.md`（精简版）

### 第二步：粘贴给 AI

打开你常用的 AI（Claude/ChatGPT/通义千问等），粘贴刚才复制的内容，发送。

### 第三步：回答问题开始写

AI 会问你：
```
1. 题材选择：玄幻/仙侠/都市/科幻/历史/游戏？
2. 主角设定：性别、性格、金手指类型？
3. 基调风格：纯爽文/轻松搞笑/热血燃向？
4. 特殊要求：后宫/单女主/无CP？
```

回答完，AI 就会输出**大纲 + 前三章**，确认后继续写。

## 📝 日常写作流程

```
每次写作：
1. 打开 AI 对话
2. 发送"继续"
3. AI 输出下一章
4. 复制章节内容，保存到 你的书名/章节/ 文件夹
5. 重复...

每写完一卷（约30万字）：
1. 发送"存档"
2. AI 输出存档信息
3. 保存到 你的书名/存档/ 文件夹
4. 下次新对话时，先发存档内容再说"继续"
```

---

## 💡 常用指令

| 你发送 | AI 会做 |
|--------|---------|
| `继续` | 写下一章 |
| `1` | 写下一章（简写） |
| `存档` | 生成当前进度存档 |
| `检查进度` | 显示字数和完成百分比 |
| `加爽点` | 下一章安排装逼打脸 |
| `扩展世界观` | 引入新地图/新势力 |
| `一致性检查` | 输出当前设定状态 |
| `这章重写` | 重新写这一章 |

---

## 🔄 换新对话时怎么续写？

AI 有对话长度限制，写一段时间后需要开新对话。

**步骤：**
1. 在旧对话里发"存档"
2. 复制 AI 输出的存档内容
3. 开新对话
4. 先发完整提示词（或精简版）
5. 再发：
```
【读取存档】

（粘贴存档内容）

从第 X 章继续写。
```

## ❓ 常见问题

### Q：提示词太长，AI 说超出限制怎么办？
A：用 `QUICK_START.md` 精简版，核心功能都有。

### Q：AI 写到 10 万字就想结局怎么办？
A：提示词里有阶段锁定系统，会自动阻止。如果还是想结局，发：
```
停！检查阶段锁定。当前字数不到 280 万，禁止写结局！继续展开新剧情！
```

### Q：前后情节对不上怎么办？
A：发"一致性检查"，AI 会输出当前所有设定，对照检查。

### Q：写出来不够好怎么办？
A：发"这章重写，参考【XXX】的风格"，指定你喜欢的作者。

## 📁 文件保存建议

```
my-novel/
├── 你的书名/
│   ├── 章节/
│   │   ├── 001-开局.md
│   │   ├── 002-金手指.md
│   │   ├── 003-第一次打脸.md
│   │   └── ...
│   ├── 存档/
│   │   ├── 第一卷存档.md
│   │   ├── 第二卷存档.md
│   │   └── ...
│   └── 设定/
│       ├── 角色档案.md（填好的）
│       └── 世界观.md（填好的）
```

每次写完就保存，别等写多了再保存，防止丢失。
