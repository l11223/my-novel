# 《摸金天师》第一卷：县城风云

> 字数：0-30万（约100章，每章3000字）
> 地图：清河县及周边
> 主角境界：土夫子 → 金点子 → 摸金手
> 核心冲突：翻身复仇 + 初入盗墓圈

---

## 一、卷概述

### 剧情主线
1. 主角被导师陷害退学，回到老家
2. 摸金符觉醒，获得地脉神眼
3. 探索县城古墓，与黑市势力周旋
4. 组建队伍，结识双女主
5. 发现明代藩王墓，多方势力争夺
6. 卷末：名声初起，进入更大的江湖

### 本卷任务
- [x] 展示金手指魅力
- [x] 完成第一个打脸循环
- [x] 组建核心队伍
- [x] 双女主出场并建立关系
- [x] 埋下长期伏笔
- [x] 打败第一个boss
