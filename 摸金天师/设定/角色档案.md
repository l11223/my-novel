## 一、主角

### 陈北辰

**基础信息**
| 项目 | 内容 |
|------|------|
| 姓名 | 陈北辰 |
| 年龄 | 25岁 |
| 身高 | 178cm |
| 外貌 | 相貌普通，眼神深邃，看人时总带着审视 |
| 学历 | 清华考古系研究生肄业（被陷害退学） |

**背景故事**
- 祖上是清末摸金校尉陈玄机，但家道中落
- 父母在他10岁时意外身亡（实际是被害，伏笔）
- 由爷爷陈守一抚养长大
- 爷爷临终前给了他一枚残破摸金符
- 以优异成绩考入清华考古系，却被导师张德厚窃取研究成果并陷害退学

**性格特点**
| 标签 | 表现 |
|------|------|
| 腹黑算计 | 表面憨厚老实，内心步步算计，从不吃亏 |
| 隐忍深沉 | 能忍常人不能忍，但记仇，总会加倍奉还 |
| 专业至上 | 对考古知识极其自信，打脸时毫不留情 |
| 重情重义 | 对认定的兄弟讲义气，对敌人绝不手软 |
| 谨慎多疑 | 凡事留一手，不会完全信任任何人 |

**口头禅**
- "让我想想..."（实际早就想好了）
- "这件事，有点意思。"
- "你确定？"
- "那就试试。"

**说话风格**
- 话不多，但每句话都有目的
- 喜欢反问，让对方自己说出答案
- 装傻时特别像，聪明时让人心惊

**金手指**
1. **地脉神眼**：能看到地下结构、墓穴布局、机关位置
2. **摸金符传承**：获得历代摸金校尉的记忆碎片，解锁各种古术

**成长轨迹**
| 阶段 | 字数 | 境界 | 状态 |
|------|------|------|------|
| 起步 | 0-30万 | 土夫子→摸金手 | 初入江湖，立足县城 |
| 成长 | 30-80万 | 摸金手→发丘将 | 圈内成名，实力提升 |
| 崛起 | 80-150万 | 发丘将→搬山真人 | 触碰帝王陵，卷入大局 |
| 称霸 | 150-220万 | 搬山真人→卸岭宗师→镇墓天师 | 成为一方巨擘 |
| 巅峰 | 220-300万 | 镇墓天师→通天仙师 | 揭示终极真相 |

## 二、双女主

### 女主A：苏锦瑶

**基础信息**
| 项目 | 内容 |
|------|------|
| 姓名 | 苏锦瑶 |
| 年龄 | 27岁 |
| 身高 | 168cm |
| 外貌 | 冷艳知性，戴金丝眼镜，长发盘起，气质如兰 |
| 身份 | 国家考古研究所女博士，镇墓司特聘顾问 |

**背景故事**
- 苏家是传承千年的守墓人家族
- 苏家与陈家（主角家族）有古老的渊源（伏笔）
- 父亲是镇墓司高层，母亲早逝
- 从小接受精英教育，26岁拿到博士学位
- 被镇墓司特聘为顾问，负责处理古墓异常事件

**性格特点**
| 标签 | 表现 |
|------|------|
| 外冷内热 | 表面高冷拒人千里，实则心软容易被打动 |
| 傲娇 | 嘴硬心软，明明关心却说"谁在乎你" |
| 学术派 | 崇尚科学，最初不信诡异，后来转变 |
| 原则性强 | 有自己的底线，不会为了目的不择手段 |
| 责任感重 | 把保护文物和民众安全放在第一位 |

**口头禅**
- "这不科学。"
- "你在胡说什么？"
- "我只是...顺便而已。"
- "哼。"

**与主角关系发展**
| 阶段 | 关系 | 字数 |
|------|------|------|
| 初遇 | 对立面（官方vs盗墓） | 5万 |
| 试探 | 互相提防，偶尔合作 | 10-30万 |
| 暧昧 | 发现主角不同，逐渐信任 | 30-80万 |
| 确认 | 并肩作战，感情明朗 | 80-150万 |
| 波折 | 身份冲突，短暂分离 | 150-200万 |
| 圆满 | 解开心结，在一起 | 200万+ |

**家族秘密**（伏笔）
- 苏家守护的墓穴里有惊天秘密
- 苏家与陈家曾有婚约
- 苏锦瑶的母亲之死另有隐情

## 六、重要NPC

### 陈守一（爷爷）

- 主角的爷爷，已故

- 临终前将摸金符交给主角
- 留下许多谜团（伏笔）

- 主角的导师，窃取研究成果陷害主角
- 表面德高望重的教授
- 实际与黑市有勾结
- 后期揭示更大阴谋

### 苏正阳（女主A父亲）

- 镇墓司高层
- 对主角态度复杂
- 知道两个家族的秘密

- 唐门当代掌门
- 豪爽江湖气
- 暗中帮助主角
