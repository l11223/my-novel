# 《摸金天师》世界观设定

---

## 一、时代背景

本书设定在**现代中国**，时间线为202X年。

表面上，中国的文物考古工作由国家文物局统一管理，各省设有考古研究所。但在地下，存在着一个庞大的盗墓江湖，延续千年。

更深层次的秘密是：古墓中存在真实的诡异力量，华夏文明的起源远比历史记载的更加神秘。

---

## 二、势力架构

### 【官方势力】

#### 1. 国家文物局
- 职能：文物保护、考古发掘、打击盗墓
- 特点：正规军，装备精良，但受限于法规

#### 2. 各省考古研究所
- 职能：地方考古工作
- 内部情况：有正派学者，也有腐败官员

#### 3. 镇墓司（秘密部门）
- 性质：国家级秘密机构，直属最高层
- 职能：处理古墓中的诡异事件，镇压邪物
- 历史：源自汉代，历代延续
- 人员：由各派传人、特殊能力者组成
- 等级：
  - 普通探员
  - 银章探员
  - 金章探员
  - 镇墓使
  - 大镇墓使

### 【地下势力——四大盗墓流派】

#### 1. 摸金派
- 历史：源自三国曹操设立的摸金校尉
- 特点：重技术，讲究风水定穴、分金定穴
- 标志：摸金符
- 规矩：
  - 鸡鸣灯灭不摸金
  - 人点烛，鬼吹灯
  - 只取一物，不可贪婪
- 现状：最正统的流派，传人稀少

#### 2. 发丘派
- 历史：与摸金派同源，但独立发展
- 特点：重传承，有发丘天印
- 标志：发丘印
- 现状：几乎绝迹，只有零星传人

#### 3. 搬山派
- 历史：源自西域，后传入中原
- 特点：不择手段，暴力开墓
- 技术：搬山分甲术，擅长破除机关
- 现状：有完整传承，但人数不多

#### 4. 卸岭派
- 历史：源自民间盗墓贼的联盟
- 特点：人多势众，靠人海战术
- 分支：唐门、蜀中派等
- 现状：人数最多，但内部松散

### 【灰色势力】

#### 1. 黑市
- 各地都有地下文物交易市场
- 分级：县级、省级、全国级
- 规矩：不问来历，只论真假

#### 2. 古董商人
- 明面上合法经营
- 暗中收购盗墓出土文物
- 有的是中间人，有的是幕后金主

#### 3. 国际势力
- **暗影组织**：国际盗墓集团，活跃于全球
- **各国博物馆**：暗中收购中国文物
- **私人收藏家**：一掷千金，不问来源

## 三、境界体系

### 【摸金校尉八境】

| 境界 | 能力描述 | 对应字数 |
|------|----------|----------|
| **土夫子** | 掌握基础盗墓技术：挖土、识别墓葬类型、基本防护 | 0-10万 |
| **金点子** | 能看风水、定穴位，判断墓葬规模和朝代 | 10-30万 |
| **摸金手** | 精通机关术、识别陪葬品真伪价值、初步感知危险 | 30-60万 |
| **发丘将** | 能感知墓中凶险、判断诡异程度、初步应对灵异 | 60-100万 |
| **搬山真人** | 可破解古墓禁制、强行开启封印、压制低级邪物 | 100-150万 |
| **卸岭宗师** | 能与墓中存在交涉、控制部分诡异力量 | 150-200万 |
| **镇墓天师** | 可镇压墓中邪物、封印诡异、修复龙脉 | 200-250万 |
| **通天仙师** | 窥探上古秘密、接触仙人遗迹、触及文明起源 | 250-300万 |

### 【境界提升条件】

1. **经验积累**：探索足够数量的古墓
2. **传承领悟**：获得前辈传承，领悟古术
3. **机缘突破**：特殊事件触发突破
4. **宝物辅助**：特殊宝物可加速提升

## 四、金手指设定

### 【地脉神眼】

**来源**：摸金符觉醒后激活的血脉能力

**能力分级**：

| 等级 | 能力 | 解锁条件 |
|------|------|----------|
| 一层 | 看穿地下10米，模糊轮廓 | 初始 |
| 二层 | 看穿地下30米，清晰结构 | 探索3座墓 |
| 三层 | 看穿地下50米，能看到机关 | 探索10座墓 |
| 四层 | 看穿地下100米，能看到气场 | 探索30座墓 |
| 五层 | 看穿地下500米，能看到诡异 | 探索50座墓 |
| 六层 | 看穿整座帝王陵 | 探索帝王陵 |
| 七层 | 看穿龙脉走向 | 触及龙脉 |
| 八层 | 看穿仙人遗迹 | 接触仙墓 |

**限制**：
- 使用消耗精神力
- 过度使用：流鼻血 → 头痛 → 昏迷
- 每天有使用上限

### 【摸金符传承】

**来源**：祖传残破摸金符，内含历代摸金校尉的记忆碎片

**解锁技能**：

| 技能 | 效果 | 解锁条件 |
|------|------|----------|
| 分金定穴 | 判断墓穴位置 | 初始 |
| 寻龙点穴 | 找到风水宝地 | 探索5座墓 |
| 破阵之术 | 破解阵法机关 | 探索15座墓 |
| 辟邪之法 | 抵御诡异侵蚀 | 遭遇诡异 |
| 镇魂之道 | 镇压低级邪物 | 击退邪物 |
| 通灵之秘 | 与墓主残魂沟通 | 特殊机缘 |
| 龙脉感应 | 感知龙脉节点 | 触及龙脉 |
| 仙人传承 | 接收仙人记忆 | 进入仙墓 |

**特殊能力**：
- 关键时刻可召唤前辈残魂指点
- 每次召唤消耗符文能量
- 符文能量通过探墓补充

## 五、诡异设定

### 【诡异分级】

| 等级 | 名称 | 特点 | 应对方式 |
|------|------|------|----------|
| E级 | 残念 | 墓主残留的情绪，无攻击性 | 无视即可 |
| D级 | 阴魂 | 普通鬼魂，有微弱攻击性 | 黑驴蹄子、糯米 |
| C级 | 厉鬼 | 有强烈怨念，会主动攻击 | 符箓、法器 |
| B级 | 凶煞 | 凝聚成实体，极其危险 | 高手联手 |
| A级 | 邪灵 | 有智慧的邪恶存在 | 镇墓天师级别 |
| S级 | 古神眷属 | 上古邪神的仆从 | 需要特殊手段 |
| SS级 | 邪神分身 | 上古邪神的化身 | 需要龙脉之力 |
| SSS级 | 邪神本体 | 被封印的上古邪神 | 最终boss |

### 【诡异来源】

1. **墓主怨念**：死不瞑目的墓主，怨念化为诡异
2. **殉葬之灵**：被殉葬者的怨恨
3. **守墓之物**：专门设置的守墓邪物
4. **上古封印**：被封印在墓中的远古存在
5. **龙脉污染**：龙脉被污染后产生的邪气

## 六、历史背景

### 【华夏文明真相】（核心伏笔，后期揭示）

**官方历史**：华夏文明起源于黄河流域，延续五千年。

**真实历史**（逐步揭示）：
1. 第一层真相（100万字揭示）：历代帝王陵墓镇压着上古邪物
2. 第二层真相（200万字揭示）：华夏文明的创立者并非普通人类
3. 第三层真相（280万字揭示）：存在"仙人"，他们创造了华夏文明后消失
4. 最终真相（300万字）：归墟——仙人的归宿之地，文明起源的终极秘密

### 【重要历史节点】

- **商周时期**：第一批邪物被封印
- **秦始皇**：建立龙脉镇压体系
- **汉武帝**：设立摸金校尉
- **唐太宗**：扩建镇墓体系
- **明太祖**：重整四大流派
- **清末民初**：传承断裂，各派衰落
- **现代**：镇墓司秘密重建

## 七、地理设定

### 【龙脉体系】

中国有三条主龙脉：
1. **北龙**：昆仑山→燕山→长白山
2. **中龙**：昆仑山→秦岭→泰山
3. **南龙**：昆仑山→横断山→五岭

### 【重要墓葬分布】

| 地区 | 重要墓葬 | 特点 |
|------|----------|------|
| 陕西 | 秦始皇陵、汉唐帝陵 | 龙脉枢纽 |
| 河南 | 商周大墓、东周王陵 | 最古老的封印 |
| 北京 | 明十三陵、清陵 | 近代帝陵 |
| 江南 | 南宋皇陵、吴越国陵 | 隐藏的地下宫殿 |
| 四川 | 三星堆、蜀王墓 | 异族文明 |
| 昆仑 | 上古仙人遗迹 | 终极秘密 |
