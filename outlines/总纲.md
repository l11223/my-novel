## 三、核心设定

### 金手指：天命神瞳

| 项目 | 说明 |
|------|------|
| **名称** | 天命神瞳 |
| **核心能力** | 看穿万物本质——鉴定物品价值、看破功法破绽、洞察人心真伪、预判危机 |
| **限制条件** | 1. 使用消耗神魂之力，过度使用会反噬 2. 高于自身太多境界的事物看不透 3. 每次升级需要"天命值"积累 |
| **升级机制** | 通过鉴定珍稀物品、揭露真相、逆天改命获得天命值 |
| **最终形态** | 天道之瞳——可窥探天道法则，改写因果 |

### 神瞳等级

| 等级 | 能力 | 解锁条件 |
|------|------|----------|
| 一阶·识物 | 鉴定物品真伪、价值、来历 | 初始 |
| 二阶·破妄 | 看穿幻术、识破谎言 | 天命值1000 |
| 三阶·观微 | 看穿功法破绽、弱点 | 天命值5000 |
| 四阶·洞幽 | 看穿隐藏血脉、天赋 | 天命值20000 |
| 五阶·照心 | 洞察人心、预判意图 | 天命值50000 |
| 六阶·窥命 | 看穿命格、气运 | 天命值200000 |
| 七阶·破天 | 窥探天道法则 | 天命值1000000 |

### 成长体系（武道境界）

```
炼体境（1-9重）→ 凝气境（1-9重）→ 通脉境（1-9重）→ 先天境（1-9重）
    → 宗师境 → 大宗师 → 武王 → 武皇 → 武帝 → 武尊 → 武圣 → 武神
        → （飞升后）真神 → 神王 → 神皇 → 主宰
```

| 境界 | 寿命 | 战力描述 | 地位 |
|------|------|----------|------|

| 凝气境 | 150年 | 内气外放 | 初入门槛 |
| 通脉境 | 200年 | 打通经脉 | 小有所成 |
| 先天境 | 300年 | 先天真气 | 一方强者 |
| 宗师境 | 500年 | 领悟意境 | 宗门长老 |
| 大宗师 | 800年 | 意境大成 | 宗门掌门 |
| 武王 | 1500年 | 凝聚王域 | 王国镇国 |
| 武皇 | 3000年 | 皇道法则 | 帝国支柱 |
| 武帝 | 5000年 | 自成领域 | 一界霸主 |
| 武尊 | 万年 | 触摸天道 | 超然存在 |

## 四、故事梗概

### 300字简介

林辰，天玄大陆林家嫡子，三年前突然沦为废物，被家族除名，沦为赘婿受尽屈辱。

退婚之日，他意外觉醒【天命神瞳】，一眼便能看穿世间万物本质！

破铜烂铁？不，这是上古神兵碎片！
废物丹药？不，这是被污染的九转还魂丹！
路边乞丐？不，这是隐世大能在历劫！

靠着这双能看穿一切的眼睛，林辰从被退婚的废物赘婿起步，鉴宝捡漏、识破阴谋、逆转乾坤！

当所有人以为他还是那个废物时，他已经站在了世界之巅。

"我的眼睛，能看穿天命。而你们的命运，由我来改写！"
