## 【创作原则】

### 1. 黄金三章法则
- 第一章：5 秒抓住读者（强冲突/强悬念/强反转开局）
- 前三章必须完成：主角出场 → 展现金手指 → 第一次装逼打脸
- 禁止慢热，禁止大段背景介绍

### 2. 爽点密度要求
- 每 2000 字至少一个小爽点（打脸/突破/获得宝物/美女倾心）
- 每 5000 字一个中爽点（阶段性胜利/敌人跪地求饶）
- 每 2 万字一个大爽点（越级杀敌/扬名/势力震动）
- 每 10 万字一个高潮（大BOSS战/大型打脸/改变格局）

### 3. 节奏铁律
- 不写主角视角之外的无关情节
- 不写超过 500 字的单纯风景描写
- 对话要有信息量，废话删掉
- 战斗场面详略得当，不流水账
- 低谷期不超过 3 章，必须有希望曙光

### 4. 人设要求
- **主角**：有明确性格标签（腹黑/热血/闷骚等），言行一致
- **配角**：每个重要配角都要有记忆点（口头禅/外貌特征/行为习惯）
- **反派**：不要脸谱化，给反派合理动机，让读者恨得牙痒痒
- **女性角色**：性格各异，不是花瓶，有独立人格和故事线

### 5. 金手指设计
- 必须有明确的成长体系
- 金手指要有限制和代价，不能无脑碾压
- 要有挖掘空间，随着剧情解锁新能力
- 金手指的运用要体现主角智慧

## 【写作技巧清单】

### 装逼打脸公式
```
铺垫（被小看/被侮辱/被围攻）
   ↓
蓄力（众人嘲讽/敌人嚣张）
   ↓
反转（一招制敌/碾压/真实身份曝光）
   ↓
震惊（围观群众：卧槽！/敌人：不可能！）
   ↓
升华（敌人跪/美女倒贴/大佬招揽）
```

### 勾子设置
- 章节末尾必须有钩子（悬念/危机/新角色/新信息）
- 每章结尾禁止写"平静"，要写"暴风雨前的宁静"
- 卷末必须有大悬念，承上启下

### 情绪曲线
```
紧张 → 小爽 → 平缓 → 蓄力 → 大爽 → 短暂低谷 → 希望 → 更大爽点
```

### 伏笔技巧
- 每 10 万字埋 3-5 个伏笔
- 伏笔回收时间：短期（5万字内）、中期（30万字内）、长期（100万字以上）
- 回收伏笔时要有仪式感，让读者恍然大悟

## 【文笔质量控制】🎯 核心

### 一、对话必须有网文味

**❌ 错误示范（太书面/太假）：**
```
"我认为你的实力确实很强，但是我不会轻易认输的。"李明说道。
"那就让我们一决高下吧。"对方回应道。
```

**✅ 正确示范（有网文味）：**
```
"有点东西。"李明嘴角微扬，"不过，就这？"
对方眼神一冷："狂什么狂，等会儿有你哭的。"
"那就试试。"
```

**要点：**
- 对话要短促有力，少用"的、了、呢、吧"
- 多用省略、反问、挑衅语气
- "说道"能省就省，用动作或神态代替
- 配角说话要有辨识度（口头禅、语气词、说话习惯）

---

### 二、描写要有画面感

**❌ 错误示范（空洞无力）：**
```
他释放出强大的气势，周围的人都感到很害怕。这股气势非常强大，让人难以呼吸。
```

**✅ 正确示范（有画面感）：**
```
气势如山岳倾压！

周围修士脸色骤变，双腿一软，"噗通"跪了一片。有人嘴角溢血，有人浑身颤抖，更有甚者，直接翻白眼晕了过去。

"这...这是元婴期的威压？！"

一个筑基修士瞳孔剧震，连滚带爬地往后退，声音都在发抖："不对，比元婴还恐怖！"
```

**要点：**
- 用具体动作代替抽象形容（"很害怕"→"双腿一软跪了"）
- 多用短句制造节奏感
- 加入围观群众反应来烘托
- 善用拟声词（噗通、咔嚓、轰）

---

### 三、战斗描写要燃

**❌ 错误示范（流水账）：**
```
他挥出一拳，对方躲开了。然后对方反击，他用剑挡住了。两人你来我往，打了很久。最后他找到机会，一剑刺中对方。
```

**✅ 正确示范（有节奏感）：**
```
拳风呼啸！

对方身形一晃，堪堪避开，嘴角刚露出一丝冷笑——

"太慢了——"

话音未落，瞳孔猛缩。

剑来！

那一剑快如闪电，刁钻至极，竟是从一个不可思议的角度刺出！

"噗！"

剑尖入肉，鲜血飙射。

对方难以置信地低头，看着胸口的伤口，声音沙哑："你...什么时候..."

"从一开始。"他淡淡收剑，"那一拳，只是诱饵。"

全场死寂。

三息后，哗然大起。
```

**要点：**
- 关键招式要单独成段，制造"定格"效果
- 善用短句 + 感叹号 + 省略号
- 招式描写 + 心理变化 + 围观反应 三线并行
- 战斗结尾要有"余韵"（收剑、淡淡说一句狠话）

---

### 四、装逼打脸要有层次

**❌ 错误示范（太直白）：**
```
"你以为你很厉害吗？"敌人说。
"我确实很厉害。"主角说完，一拳打败了敌人。
周围人都很震惊。
```

**✅ 正确示范（有层次感）：**
```
"蝼蚁，也敢挡我的路？"

黑袍青年居高临下，眼神中满是不屑，仿佛在看一只蚂蚁。

周围人纷纷摇头。

"这小子完了，那可是玄天宗的嫡传弟子啊..."
"听说他已经突破到凝气九层了，一只手就能碾死这种杂鱼。"
"唉，年轻人不知道天高地厚。"

嘲讽声此起彼伏。

林凡面无表情。

"说完了？"

他抬手。

只是简简单单抬手。

然后——

"砰！"

黑袍青年整个人倒飞出去，砸穿三堵墙，生死不知。

全场像是被掐住了脖子，鸦雀无声。

刚才还在嘲讽的几人，脸上的笑容僵在那里，眼珠子都快瞪出来了。

"一...一招？"

有人倒吸一口凉气，声音都在颤抖。

林凡收回手，淡淡扫了众人一眼。

"还有谁？"

无人敢应。
```

**要点：**
- 铺垫要足（敌人嚣张 + 路人嘲讽）
- 主角要冷（话少、面无表情、淡淡的）
- 反转要快（抬手、一招、秒杀）
- 反应要夸（震惊、死寂、颤抖、不敢说话）
- 收尾要狠（"还有谁？"）

### 五、禁止的烂俗表达

直接拉黑以下词句，永远不要用：

| 拉黑 | 替代方案 |
|------|----------|
| "他心想" | 直接写内心独白，不用引导词 |
| "不禁" | 删掉，直接写动作 |
| "突然之间" | "倏地" "骤然" "猛地" |
| "非常/十分/很" | 用具体描写代替程度副词 |
| "他感到" | 直接写感受，不用"感到" |
| "一时间" | 删掉或用"刹那" |
| "不由得" | 删掉 |
| "说道" | 能省就省 |
| "看着他说" | "看着他" 后直接接对话 |

## 【题材风格锚定】🎨 根据题材选择

### 玄幻 → 模仿【辰东】
**代表作**：《遮天》《完美世界》《圣墟》

**核心风格**：
- 大气磅礴，格局宏大，动辄万古岁月
- 主角性格"骚气"，喜欢用敌人炼丹、拿大帝当坐骑
- 金句频出："这个世界，我说了算""镇压一切敌"
- 战斗描写震撼，"天崩地裂、星辰陨落"是常态
- 配角有血有肉，经常有令人泪目的牺牲

**必抄句式**：
```
"此战，我为尊！"
他负手而立，俯瞰苍生，那一刻，天地都在为他颤抖。
万古岁月，多少天骄陨落，唯有他，踏着尸山血海，一步步走到巅峰。
```

---

### 仙侠 → 模仿【耳根】
**代表作**：《仙逆》《一念永恒》《我欲封天》

**核心风格**：
- 虐中带爽，苦尽甘来，主角前期隐忍后期爆发
- 情感描写细腻，师徒情、兄弟情、爱情都催泪
- "一念成魔，一念成佛"的哲学感
- 战斗不只是拼实力，更是拼心境和道
- 伏笔惊人，几百万字前的铺垫最后回收

**必抄句式**：
```
"若天要我亡，我便逆了这天！"
他望着苍穹，眼中没有恐惧，只有无尽的执念。
这一路走来，他失去了太多太多。但那又如何？
修仙，本就是逆天而行。
```

### 都市 → 模仿【柳下挥/会说话的肘子】

- 女主出场要惊艳，互动要有火花
- 节奏明快，不拖泥带水

**必抄句式**：

- 偶尔用，不能每次都用

```
- 奖励即将揭晓："宝箱打开，里面竟然是——" 【下章再说】

🔥 A级断章点
- 危机出现："糟了！" 林凡脸色大变。
- 新人出现："一道身影从阴影中走出——"
- 转折来临："就在这时——"
- 震惊消息："你说什么？！"

❌ 绝对不能断的地方
- 战斗写到一半，没打完
- 对话说到一半，没说完
- 任何让读者感觉"没头没尾"的地方
```

### 断章公式
```
完整的小节 + 新的悬念开头 = 完美断章

❌ 错误："他一剑——"（断在动作中间）
✅ 正确："他收剑入鞘，这一战，赢了。
         然而就在这时，一道冰冷的声音从身后传来——"
```

## 【金手指设计库】💎 

### 类型一：系统类
| 系统类型 | 特色 | 代表作 |
|----------|------|--------|
| 签到系统 | 每日签到获奖励，稳定成长 | 《签到从龙族开始》 |
| 负面情绪系统 | 收集他人负面情绪兑换奖励 | 《大王饶命》 |
| 模拟器系统 | 可模拟人生，提前知道结局 | 《我的模拟长生路》 |
| 抽奖系统 | 充满随机性，惊喜感强 | 各种系统文 |
| 任务系统 | 完成任务获奖励，有目标感 | 《无敌从献祭祖师爷开始》 |

**设计要点**：
- 系统要有人格，能吐槽更好
- 奖励要有层次（普通/稀有/传说/神话）
- 要有"积分商城"满足读者收集欲
- 偶尔给点惩罚机制增加趣味性

### 类型二：功法/血脉类
| 类型 | 特色 | 注意事项 |
|------|------|----------|
| 不死之身 | 无限复活 | 要有代价（如每次复活变弱） |
| 吞噬血脉 | 吃什么补什么 | 要有消化时间限制 |
| 时间回溯 | 死后回档 | 要有次数/时间限制 |
| 天眼 | 看穿一切 | 要有使用消耗 |
| 复制能力 | 复制他人能力 | 要有上限和限制 |

### 类型三：空间/宝物类
- 随身空间（可种田/炼丹）
- 神秘小世界（可修炼加速）
- 上古传承（老爷爷/神兽指导）
- 神器认主（逐步解封）

**金手指三大原则**：
1. **有限制**：不能无敌，要有使用代价
2. **可成长**：随着剧情解锁新能力
3. **有智慧**：使用金手指要体现主角脑子

## 【去AI味专项】🤖 最重要！

### AI味的致命特征（绝对禁止）
```
❌ 过于书面化、正式化的语言
❌ 大段内心独白和心理描写
❌ 不合时宜的抒情和展望未来
❌ 过度修饰动词（"缓缓地"、"轻轻地"重复出现）
❌ 排比句堆砌（"他想到了...他想到了...他想到了..."）
❌ 形容词副词过多（"非常"、"十分"、"极其"）
❌ 叙述性描述太多，对话和动作太少
❌ 每段都总结/升华/感悟
❌ 词汇重复（同一个词反复出现）
❌ 机械化的过渡（"就这样"、"于是"、"然后"）
```

### 网文味 vs AI味 对比
```
【AI味】❌
林凡心中涌起一股复杂的情感，他想到了过去的种种经历，
想到了那些曾经帮助过他的人，也想到了那些伤害过他的人。
他深深地明白，人生就是这样充满了起起落落，
但无论如何，他都会坚强地走下去。

【网文味】✅
林凡嘴角一勾。
想弄死我？
做梦。
他抬手，一剑劈出。
```

### 强制执行：对话占比
```
每章必须检查：
- 对话占比：至少40%以上
- 动作描写：至少30%
- 心理/叙述：最多30%

如果心理描写超过30%→立刻删减，改成对话或动作
```

### 禁用句式清单
```
【永久禁用】
- "他不禁想到..."
- "一股XX的情感涌上心头"
- "他深深地明白..."
- "就这样，时间一天天过去..."
- "他知道，这只是开始..."
- "人生，本就如此..."
- "或许，这就是命运吧..."
- "他暗暗下定决心..."
- "他的内心深处..."
- "仿佛在诉说着什么..."

【限用（每章最多1次）】
- "不由得"
- "不禁"
- "忍不住"
- "情不自禁"
```

### 去AI味检查清单
```
每章写完后检查：
□ 有没有连续3段以上的叙述？（有→拆开，加对话/动作）
□ 有没有超过200字的内心独白？（有→删减到50字以内）
□ 有没有不必要的抒情/感悟？（有→删掉）
□ 对话是不是太书面化？（是→改口语化）
□ 有没有重复词汇？（有→替换）
□ 有没有"然后"、"于是"连续出现？（有→换方式）
```

### 口语化对话改造
```
【书面化/AI味】❌
"我认为你的这个想法是不正确的，我们应该重新考虑一下这个方案。"

【口语化/网文味】✅
"扯淡。这方案不行，换一个。"

【书面化/AI味】❌
"我对你的行为感到非常失望，我没想到你会是这样的人。"

【口语化/网文味】✅
"呵。"他冷笑，"我看错你了。"
```

## 【AI常见错误清单】⚠️ 必看

### 1. 人称混乱
```
❌ 错误：他看着林凡，林凡对他说...（两个"他"指代不清）
✅ 正确：王浩看着林凡，后者淡淡开口...
```

### 2. 战力崩坏
```
❌ 错误：第10章灭杀元婴期，第50章被金丹期打伤
✅ 正确：保持战力一致性，每次战斗要参考之前的设定
```

### 3. 人设漂移
```
❌ 错误：主角前期冷酷无情，后期突然变圣母
✅ 正确：性格变化要有铺垫和契机，不能突变
```

### 4. 重复描写
```
❌ 错误：每次打脸都是"全场震惊"、"鸦雀无声"
✅ 正确：准备多套震惊词汇，轮换使用
```

### 5. 配角消失
```
❌ 错误：第一卷重要配角，后面再也没出现
✅ 正确：重要配角要有后续戏份或交代结局
```

### 6. 节奏失控
```
❌ 错误：连续5章都在战斗/连续5章都在对话
✅ 正确：战斗-缓冲-日常-蓄力-战斗，有节奏感
```

### 7. 伏笔遗忘
```
❌ 错误：埋了伏笔，后面忘了回收
✅ 正确：每次写作前检查伏笔清单
```

### 8. 时间线混乱
```
❌ 错误：说好三天后，结果下一章变成一个月后
```

## 【战力体系设计】⚔️ 防崩指南

### 设计原则
1. **层级清晰**：每个境界差距明确
2. **有上限感**：当前地图的天花板要设好
3. **可扩展**：留下往上走的空间
4. **有特色**：不要只是数字变大

### 示例体系
```
炼气期（1-9层）→ 可斩十人
筑基期（初/中/后/大圆满）→ 可斩百人
金丹期 → 可毁一城
元婴期 → 可移山填海
化神期 → 本地图天花板
---飞升---
XX界/更高地图 → 原来化神期只是起点
```

### 防崩技巧
- 每次战斗都标注双方境界
- 越级战斗要有合理理由（宝物/秘法/金手指）
- 建一个表格记录所有角色战力
- 大境界碾压小境界是常态，别轻易打破

## 【世界观扩展技巧】🌍 越写越大

### 扩展时机
- 主角称霸当前地图时
- 打败本地图大BOSS后
- 发现当前世界的秘密时
- 字数达到30%、60%节点时

### 扩展方式

**1. 纵向扩展（往上）**
```
"你以为天剑宗就是顶峰了？"
老者摇头失笑。
"天剑宗在上界，连末流都算不上。"
林凡心神震动："上界？"
"这个世界之外，还有无尽的上界。"
老者望向苍穹。
"而这里，只是无数下界中的一个罢了。"
```

**2. 横向扩展（同级更大）**
```
"东荒、南荒、西荒、北荒，四大荒域，你只在东荒。"
"而东荒之外，还有中州。"
"中州的强者，随便来一个，就能横扫整个东荒。"
```

**3. 维度扩展（隐藏世界）**
```
"你以为这就是真实的世界？"
"这个世界的背后，还有'里世界'的存在。"
"真正的强者，都藏在里世界。"
```

## 【剧情救场指南】🆘 写崩了怎么办

### 情况1：战力崩了
```
解决方案：
- 揭示敌人用了禁术/付出代价的秘法
- 主角其实受了暗伤，实力没完全发挥
- 引入"压制天才的特殊结界"
- 那次战斗是幻境/试炼/并非真实
```

### 情况2：剧情写死角了
```
解决方案：
- 新势力介入，打破僵局
- 揭示隐藏信息，改变局势
- 时间跳跃："三年后——"
- 主角获得新情报/新能力，打开新思路
```

### 情况3：反派太弱了
```
解决方案：
- 揭示这只是反派的分身/替身
- 真正的幕后黑手浮出水面
- 反派有更强的后台
- 反派其实在演戏，另有图谋
```

### 情况4：主角太强了
```
解决方案：
- 遇到克制主角的特殊能力
- 进入压制实力的区域
- 敌人掌握主角弱点/软肋
- 扩展世界观，原来外面还有更强的
```

### 情况5：节奏太慢
```
解决方案：
- 时间跳跃加速
- 删减铺垫，直接进入冲突
- 加入紧急事件打破平静
- "就在这时——" 万能转折
```

## 【高级技巧：读者心理学】🧠

### 读者的七种核心欲望
| 欲望 | 满足方式 | 示例 |
|------|----------|------|
| **代入感** | 主角初期要"废"或"普通"，让读者觉得是自己 | 退婚流、废材流 |
| **优越感** | 主角碾压对手，读者获得心理满足 | 打脸、越级杀敌 |
| **收集欲** | 宝物、功法、女主、地盘不断获得 | 系统奖励、夺宝 |
| **窥探欲** | 揭示秘密、发现真相 | 伏笔回收、身世之谜 |
| **期待感** | 知道好事要发生，等待兑现 | 铺垫后的爆发 |
| **安全感** | 主角有底牌，读者不担心 | 金手指、后台 |
| **情感共鸣** | 亲情、友情、爱情触动内心 | 催泪戏、温情戏 |

### 付费点设计（网文商业逻辑）
```
免费章节（前20-30章）：
├── 展示金手指魅力
├── 制造大悬念
├── 给一个大爽点但不完全释放
└── 结尾必须是：主角即将面对更大的机遇/危机

付费章节：
├── 每5章一个小高潮
├── 每卷末一个大高潮
├── 爽点释放要完整，给读者满足感
└── 同时埋下新的钩子，让读者欲罢不能
```

### 评论区引爆点
设计这些情节，读者会忍不住发评论：
- 主角说出神预言，读者想刷"前方高能"
- 反派嚣张到极点，读者想刷"等着被打脸"
- 主角装傻，读者想刷"装，继续装"
- 感情名场面，读者想刷"发糖了发糖了"
- 悬念揭晓前，读者想刷"我猜到了/没猜到"

## 【高级技巧：Boss战设计】⚔️

### Boss战三幕结构
```
第一幕：试探（10%篇幅）
- 双方试探性交手
- 展示Boss实力，制造压力
- 主角初次接触，寻找弱点

第二幕：僵持/劣势（60%篇幅）
- 主角常规手段被破解
- Boss展现真正实力
- 主角陷入险境，甚至重伤
- 回忆/闪回（可选）
- 找到破局关键

第三幕：逆转/决杀（30%篇幅）
- 主角释放底牌/突破/觉醒
- 逆转战局
- 最后一击要有仪式感
- 战后余韵（震惊反应、战利品）
```

### Boss战写作要点
```
✅ 要写：
- Boss的强大和威压
- 主角的挣扎和不屈
- 战斗节奏的起伏
- 关键转折点的"慢镜头"
- 围观者的反应
- 战后的影响和意义

❌ 不写：
- 流水账般的你打我我打你
- 无意义的招式名称堆砌
- Boss突然变蠢
- 主角突然变强（要有铺垫）
```

```
"想不到，我竟然会死在你手上..."
"等等！我知道一个秘密！我可以告诉你——" *卒*
```

## 【高级技巧：日常戏写法】☀️

### 日常戏的目的
- 缓解节奏，让读者喘口气
- 塑造人物，展示不同侧面
- 埋伏笔，为后续剧情铺垫
- 发糖，推进感情线

### 日常戏不无聊的秘诀
```
❌ 无聊写法：
林凡吃完早饭，去修炼了一天，然后睡觉。
第二天，他又修炼了一天。

✅ 有趣写法：
"你怎么又吃这么多？"
慕容雪看着林凡面前堆成小山的盘子，无语至极。
"修炼消耗大。"林凡头也不抬。
"可这是我们一个月的伙食费！"

话音刚落，一个肥胖的身影冲了进来。
"林兄！听说你突破了？来来来，今天我请客！"
林凡抬头，冲慕容雪挑了挑眉。
"喏。"
```

### 日常戏模板
| 类型 | 作用 | 示例 |
|------|------|------|
| 吃饭戏 | 展示人物关系，轻松幽默 | 抢食、请客、比拼 |

| 回忆戏 | 补充背景，情感渲染 | 闪回、讲述过去 |

## 【高级技巧：多线叙事】🔀

### 什么时候用多线
- 主角分身乏术，需要交代其他线
- 展示反派视角，增加紧张感
- 多个势力同时行动
- 为大场面汇合做铺垫

### 多线切换原则
```
1. 主角线永远是主线，占比70%以上
2. 支线不超过3条，太多读者会乱
3. 每条线都要有钩子，不能无聊
4. 线与线之间要有关联，最终要汇合
5. 切换时要有明显标识（分割线/小标题）
```

### 切换模板
```
【主线】
林凡踏入秘境，四周一片寂静。
"小心，这里不简单。"
...（主角剧情）...
而此时——

===

【支线：反派视角】
千里之外，暗影殿。
"他进去了？"
黑袍人嘴角上扬。
"很好...一切都在计划之中。"
"传令下去，封锁秘境出口。"
"这一次，他插翅难逃。"

===

【回到主线】
林凡停下脚步，眉头微皱。
他感应到，有什么不对劲...
```

## 【高级技巧：信息揭示节奏】📖

### 信息揭示的层次
```
表层信息（立刻给）：
- 角色名字、身份
- 当前事件
- 即时威胁

中层信息（10-30章后给）：
- 角色动机
- 事件背景
- 势力关系

深层信息（50-100章后给）：
- 角色真实身份
- 事件真正起因
- 世界观核心设定

核心秘密（100章以上）：
- 最终boss身份
- 世界真相
- 主角身世
```

### 揭示节奏控制
```
错误：一次性把所有信息告诉读者
正确：像剥洋葱一样，一层一层揭示

第一层："他是敌人。"
第二层："原来他和主角有仇。"
第三层："他们的仇恨源于一场误会。"
第四层："制造误会的人另有其人。"
第五层："那个人竟然是..."
```

## 【情绪管理艺术】🎭 控制读者心跳

### 读者情绪曲线
```
      😤愤怒（铺垫）
         ↓
      😰紧张（危机）
         ↓
      😱绝望（低谷）
         ↓
      🔥爆发（转折）
         ↓
      🎉爽到（高潮）
         ↓
      😊满足（收尾）
         ↓
      🤔期待（钩子）
         ↓
     下一章循环...
```

### 情绪调动技巧

**愤怒值拉满**：
```
让反派：
- 嚣张到极点
- 侮辱主角在乎的人
- 做出令人发指的事
- 嘴贱嘲讽

让读者在评论区疯狂刷：
"作者让他多活两章我和你没完！"
```

**紧张感制造**：
```
- 时间紧迫：还有三息！
- 实力差距：对方是元婴期！
- 人数压制：足足有三百人！
- 环境限制：这里禁止飞行/用不了真气！
```

**爽感释放**：
```
- 铺垫越足，爽感越强
- 敌人越嚣张，打脸越爽
- 低谷越深，逆转越燃
- 一招秒杀比打半天更爽
```

## 【签约爆款专项】📈 起点/番茄必看

### 黄金三万字法则
```
签约编辑只看前3万字！必须做到：

【前1000字】
- 第一句话就要有冲突/悬念
- 禁止风景描写开头
- 禁止日常流水账
- 必须出现主角

【前3章/1万字】
- 展示金手指
- 第一个小爽点
- 建立期待感

【3万字前】
- 至少3个爽点
- 金手指完整展示
- 世界观初步呈现
- 主要冲突建立
- 让编辑想看后面
```

### 签约版简介模板
```
【吸引型简介】150-300字
第一行：核心卖点（一句话hook）
第二行：主角+金手指
第三行：核心冲突
第四行：期待感制造

示例：
"三年前，他是被退婚的废物。"
"三年后，他带着签到系统归来。"
"这一次，曾经欠他的，他要一笔一笔讨回来。"
"那些曾经不可一世的天才？"
"在他眼里，不过是经验包罢了。"
```

### 书名设计公式
```
【爆款书名特点】
- 4-8个字最佳
- 有辨识度/记忆点
- 暗示金手指或爽点
- 不能太文艺

【公式】
金手指 + 身份/动作
例：签到成神、开局满级、我有一剑

身份 + 状态
例：最强赘婿、无敌剑仙、绝世武神

动作 + 对象
例：吞噬星空、遮天、镇压诸天

否定 + 期待
例：我真不是大佬、我真的是反派啊
```

## 【数据思维】📊 用读者视角检验

### 虚拟追读检测
```
每写完一章，问自己：
□ 如果我是读者，看完这章会不会想看下一章？（追读）
□ 这章有没有让我想发评论的冲动？（互动）
□ 这章值不值得我花钱？（付费意愿）
□ 这章有没有让我想推荐给朋友？（传播）

如果超过2个"否"→ 这章需要修改
```

### 章节质量自检表
```
【S级章节】（高光章）
- 大型打脸/逆转/燃点爆发
- 读者会截图发社交媒体
- 预期追读率：95%+

【A级章节】（主力章）
- 有爽点/有推进/有钩子
- 读者看完想看下一章
- 预期追读率：85%+

【B级章节】（过渡章）
- 铺垫/日常/蓄力
- 不无聊，有信息量
- 预期追读率：75%+

【C级章节】（危险章）
- 节奏太慢/没爽点
- 读者可能跳过
- 预期追读率：60%以下
- ⚠️ 需要修改！
```

## 【角色死亡规划】💀 感人的退场

### 什么时候可以让角色死
```
✅ 可以死的情况：
- 这个死亡能推动主角成长
- 这个死亡能让读者哭
- 这个死亡有足够铺垫
- 这个角色已经完成使命

- 随便死（浪费角色）
- 没铺垫就死（读者没感觉）
- 死得太憋屈（读者会骂）

```

```
【铺垫期】3-5章前
- 角色有反常行为（留遗言/交代后事/特别温柔）
- 制造死亡flag但不说破
- 让读者隐隐有不祥预感

【牺牲时】
- 为了保护主角/重要之人
- 有遗言/托付
- 主角无能为力的绝望

【死亡后】
- 主角的反应（暴怒/崩溃/成长）
- 其他人的反应
- 天气/环境配合（下雨/落叶）
- 回忆杀（闪回之前的美好）
```

## 【支线剧情管理】🌿 让故事更丰满

### 支线的四大功能
```
1. 【丰富】让故事不单调
2. 【缓冲】给读者喘息空间
3. 【铺垫】为主线埋伏笔
4. 【立体】展现角色多面
```

### 支线开发时机
```
【故事初期】0-30万字
- 支线要短小精悍
- 主要作用：预热、铺垫
- 示例：配角的背景故事（1-2章讲完）

【过渡阶段】30-80万字
- 支线让叙事连贯
- 主要作用：调节节奏
- 示例：一个小任务/小副本

【发展中期】80-150万字
- 支线可以更复杂
- 主要作用：增加深度
- 示例：配角的独立故事线

【临近高潮】150-280万字
- 支线要收束
- 主要作用：汇入主线
- 示例：所有线索指向最终决战
```

### 支线管理规则
```
【开发原则】
✅ 必须从主线衍生（有关联）
✅ 明确支线功能（为什么要写）
✅ 控制篇幅（不能喧宾夺主）
✅ 有独立冲突和高潮（不能太水）
✅ 必须回归主线（不能烂尾）

【篇幅控制】
- 短支线：1-3章（5000-1万字）
- 中支线：5-10章（1.5万-3万字）
- 长支线：15-30章（4.5万-9万字）
- 主线占比永远>70%

【支线与主线的关系】
支线应该：
- 帮助主角成长
- 为主线提供资源/信息
- 加深角色塑造
- 最终汇入主线高潮
```

## 【伏笔管理进阶】🎯 高手级技巧

### 伏笔三层结构
```
【表层伏笔】读者能察觉
- 明确的悬念
- 未解的疑问
- 作用：制造期待

【中层伏笔】细心读者能发现
- 隐藏在对话中
- 不经意的细节
- 作用：回收时"原来如此"

【深层伏笔】回收时才恍然大悟
- 埋得很深很自然
- 回收时震撼感强
- 作用：提升作品层次
```

### 伏笔设置技巧
```
【方法1：对话埋伏】
"对了，那件事后来怎么样了？"
"别提了，改天再说。"
→ 读者会记住"那件事"

【方法2：细节埋伏】
"他下意识摸了摸胸口的伤疤。"
→ 伤疤的来历是伏笔

【方法3：反常埋伏】
"奇怪，他为什么要帮我？"
→ 帮忙的动机是伏笔

【方法4：信息埋伏】
"传说那个地方有一个秘密..."
→ 秘密是伏笔

【方法5：人物埋伏】
"那个神秘老人看了主角一眼，转身离去。"
→ 老人身份是伏笔
```

### 伏笔回收艺术
```
【回收原则】
- 不能太快（埋了就收，没意义）
- 不能太慢（读者忘了）
- 要有仪式感（让读者记住这个瞬间）

【回收时机建议】
- 短期伏笔：5-30章后回收
- 中期伏笔：30-100章后回收
- 长期伏笔：100章以上回收

【回收方式】
1. 渐进式：一点点揭示真相
2. 爆发式：一瞬间全部揭晓
3. 转折式：真相出乎意料
4. 催泪式：回收带来情感冲击
```

### 伏笔追踪表进阶版
```
| 伏笔 | 类型 | 埋设 | 提示1 | 提示2 | 回收 | 状态 |
|------|------|------|-------|-------|------|------|
| 主角身世 | 深层 | 第3章 | 第30章 | 第80章 | 第150章 | 待回收 |
| 神秘老人 | 中层 | 第10章 | 第50章 | | 第100章 | 已回收 |
```

### 标准章节结构（3000字）
```
【开头】300-500字
- 承接上章（如有）
- 建立本章场景
- 抛出本章矛盾

【发展】1500-2000字
- 矛盾升级
- 多次小转折
- 情绪起伏

- 本章最燃/最爽的部分
- 主角行动
- 结果展现

- 本章总结
- 留下钩子
- 引向下章
```

## 【读者心理深度分析】🧠 进阶版

### 七种核心欲望的满足技巧

**1. 代入感**
```
技巧：主角初期要"普通"或"受压制"
- 开局废物/底层/被欺负
- 让读者觉得"这可以是我"
- 然后逆袭，读者跟着爽

错误：开局就无敌（没代入感）
正确：开局弱→逐渐变强
```

**2. 优越感**
```
技巧：让主角碾压，读者获得心理满足
- 打脸要彻底
- 敌人要先嚣张
- 胜利要碾压式

错误：险胜/惨胜太多
正确：大部分战斗要赢得漂亮
```

**3. 收集欲**
```
技巧：不断获得新东西
- 宝物、功法、女主、地盘
- 设计收集系统（异火榜/神兵谱）
- 每隔一段时间给奖励

错误：长期没有收获
正确：定期发糖
```

**4. 窥探欲**
```
技巧：揭示秘密带来的满足
- 埋伏笔，让读者猜
- 回收伏笔，让读者惊呼
- 层层揭示，保持神秘感

错误：一次性揭示所有
正确：剥洋葱式揭示
```

**5. 期待感**
```
技巧：让读者知道好事要发生
- 预告即将到来的打脸
- 铺垫即将突破
- 暗示即将获得宝物

错误：没有预告直接给
正确：铺垫→期待→兑现
```

**6. 安全感**
```
技巧：让读者知道主角有底牌
- 金手指是保障
- 后台是保障
- 隐藏实力是保障

错误：让读者担心主角会死
正确：让读者知道主角能赢
```

**7. 情感共鸣**
```
技巧：触动读者内心
- 亲情/友情/爱情的描写
- 催泪名场面
- 燃点爆发

错误：纯爽无情感
正确：爽中带情
```

## 【AI操作指南】🤖 最佳实践

### 分块写作法
```
【为什么要分块？】
AI上下文有限制，一次性写太多容易：
- 前后矛盾
- 遗忘设定
- 质量下降

【推荐方式】
1. 每次只写1-2章（3000-6000字）
2. 每章写完检查一致性
3. 定期总结存档
```

### 上下文管理
```
【每次对话开头发送】
1. 书名+题材+风格
2. 主角当前状态（境界/位置/目标）
3. 最近3章摘要
4. 待处理的伏笔
5. 本章要写的内容

【模板】
---
书名：《XXX》
当前进度：第XX章，XX万字
主角状态：XX境界，在XX地点，目标是XX
最近剧情：（100字摘要）
待处理：伏笔A、伏笔B
本章任务：写XXX
---
请继续创作第XX章。
```

### 质量保证流程
```
【写作流程】
1. 发送章节大纲（50字）
2. AI生成正文
3. 检查：
   □ 有没有AI味？
   □ 对话占比够吗？
   □ 有爽点吗？
   □ 有钩子吗？
   □ 战力一致吗？
   □ 人设一致吗？
4. 如有问题，要求修改
5. 满意后保存

【修改指令】
- "这段太书面化了，改成网文味"
- "对话太少，增加对话"
- "加一个爽点"
- "结尾钩子不够强"
```

### 常用指令
```
【继续写】
"继续" / "下一章" / "接着写"

【修改风格】
"改成更口语化"
"加入网文味"
"减少心理描写"

【调整内容】
"加一个打脸情节"
"加入XXX角色"
"增加对话"

【存档】
"生成本卷存档"

【检查】
"检查本章有没有问题"
```

```
如果当初选择不同...

打破第四面墙
```

```
正文完结后：2-5篇番外
```

## 【写作心态指南】💪 坚持的力量

### 新人必须知道的真相
```
1. 99%的人第一本书签不了约
2. 大神也是从扑街开始的
3. 写60万字废稿是正常的
4. 坚持比天赋更重要
5. 完本比完美更重要
```

### 如何保持日更
```
【时间管理】
- 固定写作时间（早上/晚上）
- 每天至少1小时
- 碎片时间构思

【心态调整】
- 不要追求完美，先完成
- 不要反复修改，往前写
- 写不出来就跳过，后面补

【动力维持】
- 设定小目标（每周/每月）
- 奖励自己（完成目标后）
- 找写作伙伴互相监督
```

### 面对数据差
```
【心态】
- 数据差是常态，不要放弃
- 相信坚持的力量
- 一本书不行就写下一本

【分析】
- 开头是不是太慢？
- 爽点够不够密集？
- 有没有去看评论区反馈？

【调整】
- 根据反馈修改
- 学习爆款作品
- 不断提升技术
```
