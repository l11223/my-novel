# 第083章 天劫将至

陆青山在天机子对面坐下。

"阁主找晚辈来，是为了幽冥之门的事？"

"不只是幽冥之门。"天机子摇头，"是整个天下的命运。"

他挥了挥手，空中出现了一幅巨大的星图。

星图上密密麻麻地标注着各种符号和线条，陆青山看不太懂。

"这是天机星图。"天机子说，"记录着过去三千年的天象变化。"

"你看这里。"

他指向星图的一角。

那里有一颗暗红色的星星，正在缓缓移动。

"这是'天煞星'。"天机子说，"每隔三千年，天煞星就会回归一次。"

"每次天煞星回归，都会带来一场浩劫。"

"三千年前的那场浩劫，让人类几乎灭绝。"

"而现在......天煞星即将再次回归。"

陆青山的脸色变得凝重。

"这和幽冥之门有什么关系？"

"天煞星的力量，会削弱所有的封印。"天机子说，"幽冥之门开了一道缝，就是因为天煞星的影响。"

"如果不在天煞星完全回归之前加固封印，幽冥之门就会彻底打开。"

"届时，幽冥界的生物会蜂拥而入，整个人间都会变成地狱。"

陆青山沉默了。

他原以为封印幽冥子之后，一切都结束了。

没想到，真正的危机才刚刚开始。

"我能做什么？"他问。

天机子看着他，眼中闪过一丝赞许。

"你手握九件神器，是目前最有希望封印幽冥之门的人。"

"但仅凭九神归一的力量，还不够。"

"需要更强大的力量。"

"什么力量？"

"天地之力。"天机子说，"在这个世界上，有五处'天地节点'。"

"每一处节点都蕴含着天地的原始之力。"

"如果你能收集这五处节点的力量，再加上九神归一，就有可能彻底封印幽冥之门。"

"五处节点在哪里？"

天机子挥手，星图上出现了五个光点。

"东海龙宫、西域魔域、南疆毒沼、北极冰原、中原神山。"

"每一处都极其危险，有无数强大的存在守护。"

"以你现在龙视境界的修为，恐怕......"

"我去。"

陆青山没有丝毫犹豫。

"不管多危险，我都要试试。"

天机子看着他，满意地笑了。

"好。"

"不愧是摸金校尉的传人。"

"那就从最近的一处开始吧——"

"东海龙宫。"
