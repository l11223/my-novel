# 第067章 获得镇煞钟

陆青山从佛塔上走下来时，苏晚晴和胖子已经等在了下面。

"青山！你成功了！"苏晚晴的眼中闪着光芒。

"太好了！"胖子兴奋地拍着他的肩膀。

空寂大师站在一旁，脸上带着欣慰的笑容。

"陆青山，你是千年来第一个通过三道考验的人。"

"镇煞钟......终于等到了它的新主人。"

他转身朝寺庙内部走去，陆青山三人紧随其后。

穿过几道回廊，他们来到了寺庙最深处的一间密室。

密室的门上刻满了经文和符咒，散发着神圣而庄严的气息。

空寂大师伸手在门上按了几下，密室的门缓缓打开。

里面是一个不大的空间，中央放着一个石台。

石台上，一口古老的铜钟静静地躺在那里。

这口铜钟比之前见到的分身要大得多，通体呈暗金色，上面刻满了密密麻麻的符文和图案。

即使只是静静地躺在那里，它也散发着一股让人心悸的威压。

这才是真正的镇煞钟！

"拿去吧。"空寂大师说。

陆青山走上前，伸手触碰铜钟。

"嗡——"

一声轻微的钟鸣响起，一股温暖的力量从铜钟中涌出，融入了陆青山的身体。

他感觉到，自己和镇煞钟之间建立了一种奇妙的联系。

他可以感知到镇煞钟的存在，也可以调动它的一部分力量。

"镇煞钟认主了。"空寂大师点点头，"从今以后，你就是它的主人。"

陆青山郑重地将镇煞钟收起。

"谢谢大师。"

空寂大师摆摆手。

"不用谢老衲，是你自己赢得了它。"

他看向陆青山，眼神变得认真。

"陆青山，镇煞钟的力量很强大，但也很危险。"

"它是用来维持平衡的，不是用来争强斗狠的。"

"如果你用它来做违背天道的事情，它会反噬你。"

"记住了吗？"

陆青山点头。

"记住了。"

空寂大师满意地笑了。

"好。"

"你们可以走了。"

他转身朝密室外走去，走到门口时，突然停下了脚步。

"对了，还有一件事。"

"什么事？"

"昆仑山的封魔塔......守护者是老衲的师弟。"

空寂大师没有回头。

"他的性格比老衲更加古怪，考验也会更加严苛。"

"但如果你告诉他，是老衲让你去的......"

"他应该会给你一个机会。"

说完，他的身影消失在了门外。

陆青山站在原地，心中涌起一股感激。

空寂大师......不仅给了他镇煞钟，还给了他一条通往封魔塔的路。

"青山，我们该走了。"苏晚晴说。

"嗯。"

陆青山点点头，带着苏晚晴和胖子，离开了镇煞寺。
