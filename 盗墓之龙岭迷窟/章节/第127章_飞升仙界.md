# 第127章 飞升仙界

渡劫成仙后，陆青山在龙岭镇休整了三天。

这三天里，他巩固了仙人初期的境界，同时向众人告别。

"爷爷，我要去仙界了。"

"去吧。"爷爷慈祥地说，"找到你父母后，替我向他们问好。"

"我会的。"

"胖子，老陈头......你们帮我照顾好爷爷和龙岭镇。"

"放心！"胖子拍着胸脯，"有我在，谁也别想动龙岭镇一根草！"

陆青山转向苏晚晴。

"晚晴......"

苏晚晴摇摇头，打断他。

"我和你一起去。"

"什么？"

"我说，我和你一起去仙界。"苏晚晴的眼神坚定，"我不要留在这里等你。"

"我要和你一起。"

"但你还没有渡劫......"

"我知道。"苏晚晴说，"但仙界应该也有凡人能够生存的地方吧？"

"只要和你在一起，去哪里都可以。"

陆青山沉默片刻，叹了口气。

"好吧。"

他知道，苏晚晴一旦做出决定，就不会改变。

而且......他也舍不得和她分开。

"那就一起去吧。"

三天后。

龙岭山顶。

陆青山和苏晚晴站在山巅。

众人在山下送行。

"去吧！"爷爷挥手，"记得回来看看！"

"我会的！"

陆青山运转仙力，撕开了一道空间裂缝。

裂缝的另一端，是一片金色的世界。

那里......就是仙界！

"走吧。"

他牵着苏晚晴的手，迈入空间裂缝。

下一刻，两人消失在众人的视野中。

......

当陆青山再次睁开眼睛时，发现自己置身于一片云海之上。

云海下方是连绵的山脉，山脉上矗立着无数宏伟的宫殿。

空气中弥漫着浓郁的灵气，比凡间浓郁了何止千倍。

"这就是......仙界？"

苏晚晴惊叹地看着四周。

"好美......"

陆青山点点头。

"确实很美。"

"不过......我们得先找个地方落脚。"

他带着苏晚晴，朝最近的一座城池飞去。

城池名为"天元城"，是仙界的一座边缘城市。

城中熙熙攘攘，到处都是修士。

"这里的修士......最低都是金丹境界。"陆青山观察着四周，"看来仙界的修炼体系和凡间不同。"

"我们需要尽快了解这里的规则。"

两人走进城中，开始打探消息。

仙界......

他们的新旅程，正式开始。
