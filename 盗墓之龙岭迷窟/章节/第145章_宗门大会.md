# 第145章 宗门大会

大殿内聚集了数千名弟子。

他们都是这次战争的参与者。

玄天子站在高台上，表情严肃。

"各位弟子，这次战争，我们虽然没有失败，但也付出了惨重的代价。"

"三千名弟子阵亡，五千人重伤。"

"他们都是我们玄天宗的英雄。"

"让我们为他们默哀。"

众人低下头，默哀片刻。

"但战争也是成长的催化剂。"玄天子继续说，"在这次战争中，有许多弟子表现优异，展现出了强大的实力和勇气。"

"现在，我要宣布这次战争的功勋榜。"

"第一名：陆青山。斩杀妖兵上千，击败妖将十余名，功勋卓著。奖励：100万贡献点，仙器一件。"

"什么！"

众人惊呼。

100万贡献点！

仙器一件！

这是何等丰厚的奖励！

陆青山也愣住了。

他没想到自己会排名第一。

"陆青山，上来领奖。"

他走上高台，接过玄天子递来的物品。

那是一把剑。

剑身通体金黄，散发着神圣的光芒。

"这是'天罚剑'，仙器级别的神兵。"玄天子说，"它能增幅使用者的攻击力，非常适合你。"

"多谢宗主！"

陆青山接过天罚剑，感受着其中蕴含的力量。

"好强......"

他能感觉到，有了这把剑，他的战力至少能提升三成。

"第二名：凌霜。斩杀妖将五名，奖励：50万贡献点。"

"第三名：......"

功勋榜宣读完毕，大会结束。

陆青山带着天罚剑回到住处。

"天罚剑......"

他轻轻抚摸着剑身。

"以后，你就是我的搭档了。"

就在这时，苏晚晴走了进来。

"青山，恭喜你获得第一名。"

"谢谢。"

苏晚晴看着他手中的天罚剑，眼中闪过羡慕。

"好漂亮的剑。"

"是啊。"陆青山笑了笑，"有了这把剑，我的实力又能提升不少。"

"对了，晚晴，你这段时间修炼得怎么样？"

"我......"苏晚晴有些不好意思，"我刚突破到仙人初期。"

"仙人初期？"陆青山惊喜道，"太好了！"

"你也成仙了！"

苏晚晴笑了。

"是啊，我终于跟上你的脚步了。"

"虽然还差很远......"

陆青山握住她的手。

"没关系，我们一起努力。"
