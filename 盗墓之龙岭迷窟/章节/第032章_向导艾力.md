# 第032章 向导艾力

艾力的家在乌鲁木齐郊区的一个小村子里。

院子不大，但收拾得很干净。屋里挂满了各种猎物的皮毛和骨头，墙角还放着几把老旧的猎枪。

"委屈你们了。"艾力给三人倒上热茶，"条件简陋，将就一晚。"

"已经很好了。"陆青山说，"艾力大叔，您对罗布泊那边熟悉吗？"

"熟悉。"艾力点点头，"我年轻的时候，经常去那边打猎。那时候罗布泊还没完全干涸，湖边有很多野骆驼和黄羊。"

"后来呢？"

"后来湖水慢慢干了，动物也少了。"艾力叹了口气，"现在那里就是一片死地，连草都不长。"

"只有沙子、盐碱，还有......那些奇怪的东西。"

陆青山的眼神一动。

"什么奇怪的东西？"

艾力沉默了一下，似乎在犹豫要不要说。

"说吧，艾力大叔。"苏晚晴说，"我们来这里，就是为了那些'奇怪的东西'。"

艾力看了看三人，最终开口：

"三十年前，我第一次进罗布泊深处的时候，看到过一座古城。"

"古城？"

"对，古城。"艾力的声音变得低沉，"那座古城埋在沙子底下，只露出一小截城墙。城墙上刻着一些奇怪的文字，我看不懂。"

"我当时年轻，胆子大，就想进去看看。"

"结果......"

他的脸色变得有些苍白。

"结果我看到了不该看的东西。"

"什么东西？"陆青山追问。

"人。"艾力说，"很多很多的人。"

"他们穿着古代的衣服，站在城里一动不动。眼睛睁得大大的，好像在看着什么。"

"我以为他们是活人，就喊了一声。"

"结果他们全都转过头来......看着我。"

"那些眼睛......是空的。没有眼珠，只有两个黑洞洞的窟窿。"

胖子打了个寒颤。

"我操......那是什么玩意？僵尸？"

"不知道。"艾力摇头，"我当时吓坏了，转身就跑。跑了一天一夜，才逃出那片区域。"

"从那以后，我再也没进过那里。"

陆青山和苏晚晴对视一眼。

那座古城......很可能就是卸岭一脉的祖庭。

而那些"人"，应该是某种镇墓的手段。

"艾力大叔，那座古城的位置，您还记得吗？"苏晚晴问。

艾力犹豫了一下。

"记得是记得......但那地方太危险了。你们真的要去？"

"必须去。"陆青山说，"我们有非常重要的东西要找。"

艾力沉默了很久。

最终，他叹了口气。

"好吧。"

"既然收了你们的钱，我就带你们去。"

"但丑话说在前头——进了那片区域，生死自负。"

"我只负责带路，不负责救人。"

"没问题。"陆青山点头，"我们自己的命，自己负责。"

艾力看着他，眼中闪过一丝赞许。

"有种。"

"好，明天一早出发。"

"今晚好好休息，养足精神。"

"进了沙漠......就没有回头路了。"
