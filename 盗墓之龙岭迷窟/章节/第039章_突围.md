# 第039章 突围

那个拦路的幽冥会成员身材高大，手中的长刀足有一米多长，刀刃上闪着幽蓝色的光芒——显然淬了毒。

"把卸岭鼎交出来。"他冷冷地说，"否则，你们一个都走不了。"

陆青山将卸岭鼎递给苏晚晴，然后从背后抽出搬山铲。

"胖子，保护苏晚晴。"

"好！"

胖子挡在苏晚晴面前，手里握着一把匕首。

陆青山朝那人走去，眼中闪过一丝寒光。

"让开，我不想杀人。"

"哈哈哈！"那人大笑，"就凭你一个观气境界的毛头小子？我可是通玄境界！你在我面前，连三招都撑不过！"

"那就试试。"

陆青山的阴阳瞳全力运转，将那人的每一个动作都看得清清楚楚。

通玄境界......比他高一个大境界。

正面硬拼确实不是对手。

但他有阴阳瞳，可以预判对方的攻击。

只要抓住破绽......

"找死！"

那人怒喝一声，长刀横劈过来。

刀势凌厉，带着一股杀气。

陆青山侧身一闪，避开刀锋，同时搬山铲朝那人的手腕砸去。

"当！"

那人反应极快，用刀背挡住了搬山铲。

但陆青山没有停手，铲柄一转，直刺那人的咽喉。

那人脸色一变，连忙后退。

"你......"

他没想到一个观气境界的小子，竟然能和他过招。

"我说了，让开。"

陆青山的声音冰冷。

那人咬了咬牙，再次冲了上来。

这一次，他的攻击更加凌厉，刀势如暴风骤雨，几乎没有任何间隙。

陆青山的压力骤然增大。

即使有阴阳瞳的帮助，他也只能勉强躲避，根本没有还手的机会。

"差距还是太大了......"

他心中暗道。

就在这时，身后传来一声暴喝。

"去死吧！"

是胖子。

胖子不知何时绕到了那人身后，手里的匕首朝那人的后背刺去。

那人感觉到背后的危险，不得不放弃攻击陆青山，转身格挡。

"找死！"

他一刀劈向胖子。

但就在这一瞬间——

陆青山抓住了机会！

搬山铲狠狠地砸在那人的后脑上。

"砰！"

那人闷哼一声，踉跄着倒了下去。

"走！"

陆青山拉起胖子，和苏晚晴一起朝着出口狂奔。

三人穿过地下城的街道，沿着来时的洞道向上跑。

身后，宫殿里的战斗声渐渐远去。

石像虽然强大，但数量有限，撑不了多久。

一旦鬼手腾出手来追赶，他们就危险了。

"快点！快点！"

陆青山催促着两人加快速度。

终于，前方出现了一道光亮。

出口！

三人冲出洞口，阳光刺得他们睁不开眼睛。

"艾力大叔！"

艾力正蹲在不远处，看到他们出来，连忙站起身。

"你们还活着！"

"快走！"陆青山喊道，"有人追上来了！"

艾力二话不说，带着三人朝来时的方向狂奔。

沙漠里的追逐，才刚刚开始。
