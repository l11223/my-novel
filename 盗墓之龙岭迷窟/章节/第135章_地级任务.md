# 第135章 地级任务

任务殿。

陆青山来到地级任务区，仔细挑选任务。

地级任务比玄级任务危险得多，但奖励也丰厚得多。

"地级任务：调查妖域边境的异常波动。奖励：50000贡献点。"

"地级任务：护送长老前往东域。奖励：30000贡献点。"

"地级任务：探索北域深处的上古遗迹。奖励：100000贡献点。"

最后一个任务吸引了陆青山的注意。

"上古遗迹？"

他仔细查看任务详情。

"北域深处发现一座上古遗迹，据说里面藏有太古时期的宝物。"

"但遗迹中充满危险，已有三批探索队全军覆没。"

"本次任务招募实力强劲的弟子组队探索，每人奖励100000贡献点。"

100000贡献点！

这可是一笔巨款！

而且是上古遗迹，说不定里面有对他修炼有帮助的东西。

"我接这个任务。"

他在任务板上按下手印。

"任务已接取。集合时间：三天后。集合地点：宗门大门。"

三天后。

宗门大门前聚集了十几个人。

他们都是接了这个任务的弟子。

领队是一个中年男子，境界是仙王初期。

"大家都到齐了？"领队扫视众人，"好，出发。"

队伍朝北域深处飞去。

陆青山跟在队伍中，观察着同行的人。

这些人最弱的也是仙人后期，最强的是一个仙君中期的女子。

"你好，我叫陆青山。"他主动和女子打招呼。

"我叫凌霜。"女子淡淡地说，"玄天宗内门弟子。"

"你是新来的吧？没见过你。"

"是，我刚从凡间飞升不久。"

"凡间飞升？"凌霜眼中闪过一丝惊讶，"你才飞升多久，就达到了仙人中期？"

"大概两个月。"

"两个月？！"

凌霜震惊了。

两个月从仙人初期突破到仙人中期？

这速度太惊人了！

"你的天赋......很强。"她认真地看着陆青山。

"谢谢。"

两人边走边聊，很快熟络起来。

七天后。

队伍来到了北域深处。

前方是一片荒凉的山脉，山脉中央有一座巨大的石门。

"到了。"领队说，"这就是上古遗迹的入口。"

"进去之后，大家小心行事。"

"遇到危险，保命第一。"

众人点头。

领队推开石门，带着众人走了进去。
