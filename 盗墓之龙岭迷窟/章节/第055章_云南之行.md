# 第055章 云南之行

第二天一早，三人踏上了前往云南的旅程。

从龙岭镇到云南，需要先坐火车到昆明，然后再转汽车进入滇西山区。

全程大约需要三天。

火车上，苏晚晴给陆青山和胖子讲解着关于苗疆蛊术的知识。

"蛊术是苗疆的传统巫术，历史比四大盗墓门派还要古老。"

"最早的蛊术可以追溯到商周时期，距今已有三千多年。"

"蛊虫种类繁多，有金蚕蛊、蛇蛊、蜈蚣蛊、蜘蛛蛊......每一种都有不同的效果。"

"有的能控制人心，有的能毒杀敌人，有的甚至能让人起死回生。"

"最厉害的蛊虫，叫做'万毒之王'。"

"据说练成万毒之王的人，可以控制天下所有的毒虫，百毒不侵。"

"万蛊洞里的那位蛊王，据说就是练成了万毒之王的人。"

陆青山听得心惊。

蛊术......比他想象的要可怕得多。

"有办法对付蛊虫吗？"

"有。"苏晚晴点头，"火克蛊，强烈的火焰可以杀死大部分蛊虫。"

"另外，黑驴蹄子和雄黄也有驱蛊的效果。"

"我们这次带了不少雄黄粉，应该能派上用场。"

"那就好。"

火车轰隆隆地行驶着，窗外的风景从平原变成了山地，又从山地变成了丛林。

三天后，昆明。

三人下了火车，又转乘了一辆破旧的长途汽车，朝着滇西山区进发。

汽车在蜿蜒的山路上颠簸了一整天，终于在傍晚时分到达了一个小镇。

小镇名叫"蛊寨"，是进入万蛊洞的必经之路。

"今晚就在这里住下。"苏晚晴说，"明天一早进山。"

三人找了一家小旅馆，要了三间房。

晚上，陆青山独自坐在房间里，思考着接下来的行动。

万蛊洞......归墟老人......归墟令......

这一切都像是一个精心设计的局。

但他没有选择。

为了收集九件神器，为了对抗幽冥会，他必须走进这个局里。

"咚咚咚。"

敲门声响起。

"谁？"

"是我。"

苏晚晴的声音。

陆青山起身开门。

苏晚晴站在门口，手里拿着一个小瓷瓶。

"这是什么？"

"解蛊丹。"苏晚晴说，"发丘一脉的秘药，可以解除大部分蛊毒。"

"我只有三颗，你拿一颗，我和胖子各一颗。"

"万一中了蛊，就吃下去。"

陆青山接过瓷瓶，心中涌起一股暖意。

"谢谢。"

苏晚晴微微一笑。

"早点休息，明天还要赶路。"

"嗯。"

门关上，陆青山看着手中的瓷瓶，嘴角露出一丝笑容。

有这样的同伴，真好。
