## 核心设定：规则怪谈

### 什么是规则怪谈？

三年前的某一天，全球各地同时出现了"规则"。

这些规则出现在各种地点——电梯、医院、学校、便利店、公交车、老旧小区......

规则以文字形式显现，任何人都能看到。

而一旦进入规则覆盖的范围，就必须遵守规则。

**违反规则者——消失。**

不是死亡，是彻底从世界上抹去，连记忆都不会留下。

### 规则的特点

1. **规则不可物理破坏**：写有规则的载体无法被任何手段毁坏
2. **规则范围固定**：每个怪谈都有明确的边界
3. **规则会变化**：高级怪谈的规则可能随时间/条件改变
4. **规则有真假**：部分规则是"陷阱"，遵守反而会死
5. **规则有核心**：每个怪谈都有一个"核心"，破坏核心可永久消除怪谈

## 觉醒者体系

### 什么是觉醒者？

通关怪谈的人，有几率获得"异能"，成为觉醒者。

觉醒几率与怪谈等级正相关：
- D级：1%
- C级：5%
- B级：15%
- A级：40%
- S级：80%
- SS级：100%

### 觉醒者等级

| 等级 | 全球数量 | 能力特征 |
|------|----------|----------|
| F级 | 约500万 | 异能微弱，日常辅助级 |
| E级 | 约100万 | 异能实用，可战斗 |
| D级 | 约20万 | 异能强化，职业觉醒者 |
| C级 | 约5万 | 异能成熟，可单挑C级怪谈 |
| B级 | 约1万 | 异能精深，区域知名 |
| A级 | 约2000 | 异能登峰，国家级战力 |
| S级 | 约200 | 异能巅峰，传说人物 |
| SS级 | 约20 | 超越人类范畴 |

### 常见异能类型

**战斗系**
- 强化类：增强身体素质
- 元素类：操控火、水、雷等元素
- 空间类：瞬移、空间切割
- 时间类：减速、加速（极其稀有）

**辅助系**
- 感知类：增强感官
- 治愈类：治疗伤势
- 预知类：预感危险
- 分析类：信息处理能力增强

**特殊系**
- 规则类：与规则互动的能力（极其稀有）
- 怪谈类：获得怪谈的部分特性

## 规则设定示例

### 【午夜便利店】- D级

**位置**：江城老城区某便利店

**规则**：
```
1. 营业时间为23:00-05:00，其他时间禁止进入
2. 购物时不得与店员对视超过3秒
3. 若听到"欢迎光临"以外的话语，立即离开
4. 付款时只能使用现金
5. 离开时必须说"谢谢"
```

**真相**：
- 店员不是人，是怪谈的一部分
- 第3条是核心规则，违反必死
- 其他规则违反会触发惩罚但不致死

**核心**：收银台下的一本账本

---

### 【凌晨三点的电梯】- C级

**位置**：江城某老旧写字楼

**规则**：
```
1. 只能在凌晨三点使用
2. 按下楼层后，禁止更改
3. 若有人进入，不得说话
4. 若电梯停在没有按下的楼层，闭眼倒数十秒
5. 若倒数时听到耳语，不得回应
6. 到达目的地后，必须先出右脚
7. 若镜子里的你在微笑，不要恐惧
```

**真相**：
- 第4条是陷阱规则，正确做法是睁眼直视"它"
- 第7条是关键提示，镜子里的"你"其实是怪谈的本体
- 通关条件：与镜中的"自己"对视超过10秒

**核心**：电梯里的镜子

---

### 【废弃学校·七日怪谈】- A级

**位置**：江城郊区废弃中学

**规则**：（每天更新）
```
第一日：不得触碰任何课桌
第二日：必须每小时移动一次
第三日：禁止进入三楼
第四日：午夜十二点必须在操场
第五日：所有规则反转
第六日：新规则与旧规则同时生效
第七日：找出"校长"并回答它的问题
```

**真相**：
- "反转"不是字面意思，而是指"最重要的规则不再适用"
- 前四天的规则中有一条是最核心的保护规则
- 第五天开始，必须找出并违反那条规则才能存活
- "校长"是学校的核心，形态为一个戴着面具的影子
- 它的问题是："你违反了几条规则？"正确答案是实际数字+1

**核心**：校长室的一张毕业照
