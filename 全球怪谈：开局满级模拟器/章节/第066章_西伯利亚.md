# 第066章 西伯利亚

从欧洲回来后。

林默没有休息太久。

下一个目标已经确定——

【死亡森林】，位于西伯利亚的SS级怪谈。

"你确定要去？"

沈清音看着林默，眼中带着担忧。

"那里的环境极其恶劣。"

"零下四十度的严寒，无尽的雪原......"

"再加上怪谈本身的威胁......"

"我确定。"

林默收拾着行李。

"还差两枚碎片。"

"【死亡森林】很可能包含一枚。"

"我必须去。"

沈清音沉默了。

她知道劝不住林默。

这个人一旦做出决定，就不会改变。

"那我跟你一起去。"

"不行。"

林默摇头。

"这次的任务太危险。"

"西伯利亚的环境本身就是致命的。"

"再加上怪谈......"

"你跟着去只会成为累赘。"

"但——"

"听我说。"

林默看着她的眼睛。

"我需要你留在京都。"

"帮我收集最后一枚碎片的情报。"

"这同样重要。"

沈清音咬着嘴唇，最终点了点头。

"好......"

"你一定要小心。"

"放心。"

林默笑了笑。

"我会活着回来的。"

......

两天后。

西伯利亚。

林默乘坐直升机，来到了【死亡森林】的边缘。

眼前是一片无尽的白色。

雪覆盖了一切——树木、大地、甚至空气中都飘着雪花。

但在这片白色中，有一块区域是不同的。

那里的树木是黑色的。

枯死的、扭曲的、诡异的黑色。

就像是从地狱里长出来的一样。

"就是那里。"

俄罗斯调查局的联络员指着那片黑色森林。

"【死亡森林】。"

"三年前，那里本来是一片普通的针叶林。"

"直到'怪谈'降临。"

"现在......那里是全俄罗斯最危险的地方。"

林默看着那片森林。

即使隔着几公里的距离，他也能感受到一股诡异的气息。

那是"死亡"的气息。

"规则呢？"

"在森林边缘有一块石碑。"

联络员说。

"上面刻着规则。"

"但说实话......那些规则很奇怪。"

"奇怪？"

"您去看看就知道了。"

......

林默来到森林边缘。

果然，那里立着一块巨大的石碑。

上面刻着俄文，但林默可以看懂——模拟器自带翻译功能。

【死亡森林·生存须知】

```
1. 不要杀死任何东西
2. 若看到尸体，不要触碰
3. 若听到自己的名字，不要回应
4. 不要在同一个地方停留超过十分钟
5. 若遇到"活着的死人"，假装是它们的同伴
6. 不要吃森林里的任何东西
7. 若看到光，跟着它走
8. 若光消失，立刻停下
9. 不要试图离开森林——除非你找到"出口树"
10. "出口树"是唯一一棵有叶子的树
......
```

共45条规则。

"不要杀死任何东西......"

林默皱眉。

"这个规则很有意思。"

一般的怪谈，都是要求"杀死敌人"或"躲避敌人"。

但这个怪谈——不能杀死任何东西。

"难道......这里的东西都是'死而复生'的？"

"杀死它们反而会让它们变得更强？"

他决定通过模拟来验证。

"开始模拟。"

【第1次模拟开始】

......

模拟中。

林默走进森林。

周围的温度骤降。

冷，刺骨的冷。

即使穿着厚重的防寒服，他也能感受到那种寒意。

但更可怕的是——

那种"死亡"的气息。

每一棵树都是枯死的。

地上散落着各种动物的尸体——鹿、狼、熊......

还有......人类的尸体。

林默小心翼翼地绕过那些尸体。

规则2：若看到尸体，不要触碰。

他继续前进。

走了大约十分钟——

他注意到一个问题。

周围的环境......没有变化。

树还是那些树，尸体还是那些尸体。

就好像他一直在原地打转。

"空间是扭曲的......"

他皱眉。

规则4说不能在同一个地方停留超过十分钟。

但如果空间是扭曲的，他可能根本没有"移动"。

"必须找到方向。"

他看向四周。

就在这时——

他发现了一个光点。

远处，有一束微弱的光。

规则7：若看到光，跟着它走。

林默向那束光走去。

光在移动，他跟着。

走了大约五分钟——

光突然消失了。

规则8：若光消失，立刻停下。

林默停住脚步。

周围一片寂静。

然后——

他听到了脚步声。

从身后传来。

他缓缓转过身——

一具"尸体"站在他身后。

那是一个人类——或者曾经是人类。

它的皮肤是灰色的，眼睛是空洞的，嘴巴张开，露出腐烂的牙齿。

"活着的死人......"

林默心中一凛。

规则5：若遇到"活着的死人"，假装是它们的同伴。

他试着模仿"死人"的姿态——僵硬、缓慢、面无表情。

"死人"盯着他看了几秒。

然后——

它转身，慢慢走开了。

林默松了口气。

"规则5是真的......"

"只要假装是同伴，它们就不会攻击。"

他继续探索。

但很快——

他发现了一个问题。

这片森林......没有尽头。

规则9说不能离开森林——除非找到"出口树"。

"出口树"是唯一一棵有叶子的树。

但这片森林里，所有的树都是枯死的。

"'出口树'在哪......"

他继续寻找。
