# 第003章 凌晨三点的电梯

江城，老旧写字楼。

林默站在一栋灰色建筑前，抬头看着顶楼。

这里就是C级怪谈【凌晨三点的电梯】所在地。

"你确定要进去？"

沈清音站在他身后，表情严肃。

"这是C级怪谈，比你昨晚遇到的D级强了不止一个档次。"

"而且里面已经有三个觉醒者被困了。"

"他们都是有经验的老手，却到现在都没能出来。"

林默没有回答。

他在看手机上的规则资料——调查局的内部数据库。

【凌晨三点的电梯】

等级：C级

位置：江城旧城区某写字楼1号电梯

规则：
```
1. 只能在凌晨三点使用
2. 按下楼层后，禁止更改
3. 若有人进入，不得说话
4. 若电梯停在没有按下的楼层，闭眼倒数十秒
5. 若倒数时听到耳语，不得回应
6. 到达目的地后，必须先出右脚
7. 若镜子里的你在微笑，不要恐惧
```

通关条件：乘坐电梯到达任意楼层并成功离开

已知信息：部分规则可能为假

"现在是凌晨两点五十分。"沈清音看了眼手表，"还有十分钟，怪谈就会激活。"

"到时候这栋楼的所有电梯都会被'它'接管，直到有人通关或者所有人死亡。"

"三个觉醒者在里面已经待了两天了。"

"你还有机会退出。"

林默收起手机。

"不用。"

他走向写字楼大门。

"我只需要十分钟。"

沈清音："？"

她没明白林默的意思。

十分钟？

C级怪谈，十分钟能做什么？

......

林默走进写字楼。

大厅空无一人，弥漫着一股腐朽的气息。

他看向角落里的电梯——此刻还是普通的电梯，但十分钟后，它就会变成怪谈的入口。

"开始模拟。"

他在心里默念。

【是否消耗时间进行模拟？】

【提示：模拟时间与现实时间1:1流动】

【当前剩余时间：9分47秒】

"是。"

刷——

眼前一花。

再睁眼时，时间已经是凌晨三点。

电梯门缓缓打开。

林默走了进去。

【第1次模拟开始】

电梯内部很普通——按钮、镜子、照明灯。

林默按下了"5"楼。

电梯开始上升。

1楼、2楼、3楼——

"叮。"

电梯停在了4楼。

但他没有按4楼。

规则4：若电梯停在没有按下的楼层，闭眼倒数十秒。

林默闭上眼睛。

"十、九、八......"

电梯门打开了。

一阵阴冷的风吹进来。

"......你在数什么？"

一个沙哑的声音在耳边响起。

规则5：若倒数时听到耳语，不得回应。

林默没有回答，继续倒数。

"......为什么不说话？"

"......看看我......"

"......看看我啊......"

林默紧闭双眼，咬紧牙关。

"三、二、一——"

电梯门关闭。

继续上升。

5楼。

"叮。"

到了。

林默睁开眼，迈出右脚——

等等。

镜子。

他看向电梯里的镜子。

镜中的他......在微笑。

但他没有笑。

规则7：若镜子里的你在微笑，不要恐惧。

不要恐惧......

林默深吸一口气，强压下心中的不安，迈步走出电梯。

然后——

他的影子动了。

不是他在动，是影子自己在动。

影子从地面"站"了起来，化作一个漆黑的人形。

它扑向林默——

死亡。

【第1次模拟结束·死亡原因：未知】

"有问题。"

林默睁开眼，再次站在电梯门口。

"我遵守了所有规则，为什么还是死了？"

他回想刚才的情况。

规则4，闭眼倒数——遵守了。

规则5，不回应耳语——遵守了。

规则7，不恐惧镜中的微笑——遵守了。

但还是死了。

"有假规则。"

这是C级怪谈的特点——部分规则是陷阱。

"再来。"

【第2次模拟开始】

这一次，林默故意违反规则4。

电梯停在没有按下的楼层时，他没有闭眼，而是睁眼看向门外——

什么都没有。

门外是空荡荡的走廊。

没有危险。

"规则4......是假的？"

电梯门关闭，继续上升。

到达5楼。

林默迈出右脚，走出电梯。

这一次，他特意避开了镜子，没有去看。

但——

影子还是动了。

死亡。

【第2次模拟结束·死亡原因：未知】

"不是规则4的问题。"

林默皱眉。

"那就是规则7？"

"不对......规则7说'若镜子里的你在微笑，不要恐惧'。"

"这是在暗示，镜子里的'我'是敌人？"

"还是说......"

他眼中闪过一丝精光。

"镜子里的，才是'真正的我'？"

【第3次模拟开始】

这一次，林默进入电梯后，第一时间看向镜子。

镜中的他——没有微笑。

表情和他一模一样。

"正常的。"

他按下5楼，电梯开始上升。

1楼、2楼、3楼、4楼——

电梯没有停。

直接到了5楼。

"叮。"

林默看向镜子。

镜中的他......依然没有微笑。

"有意思。"

他迈出右脚，走出电梯。

影子——没有动。

他成功走出了电梯。

但......

他没有通关。

周围的场景没有变化，还是在这栋写字楼里。

"通关条件是'离开'，但我已经离开电梯了，为什么没通关？"

林默思考着。

"除非......"

"'离开'不是指离开电梯，而是离开这栋楼？"

他走向楼梯。

推开楼梯门的瞬间——

一张没有五官的脸，近在咫尺。

死亡。

【第3次模拟结束·死亡原因：尝试离开怪谈范围】

"不能走楼梯。"

林默再次睁开眼。

"只能用电梯。"

"但电梯只能在凌晨三点使用......"

"等等。"

他突然想到了什么。

"规则说'只能在凌晨三点使用'。"

"那凌晨三点零一分呢？"

"三点过后，电梯还能用吗？"

【第4次模拟开始】

这一次，林默进入电梯后，没有立刻按楼层。

他等待。

三点零一分。

三点零二分。

三点零三分——

电梯的灯闪了一下。

然后——

【系统提示：已超出使用时间，违反规则1】

死亡。

【第4次模拟结束·死亡原因：违反规则1】

"规则1是真的，只能在三点整使用。"

林默迅速整理信息。

目前确定的：
- 规则1（真）：只能在凌晨三点使用
- 规则4（假？）：闭眼倒数反而安全
- 规则6（真？）：必须先出右脚
- 规则7（关键）：镜子微笑时有问题

"让我再试试规则6和7。"

【第5次模拟开始】

林默进入电梯，按下5楼。

这一次，他全程盯着镜子。

1楼——镜中正常。

2楼——镜中正常。

3楼——镜中正常。

4楼——

镜中的他，嘴角微微上扬。

在微笑。

但林默本人没有笑。

"来了。"

规则7说"不要恐惧"。

但林默有个大胆的想法——

如果"不要恐惧"的意思，不是让你平静地忽略它......

而是让你"面对"它呢？

他盯着镜中的自己。

然后——

他也笑了。

和镜中的"自己"，做出同样的表情。

镜中——

那个"他"的笑容僵住了。

下一秒，镜面炸裂。

一团漆黑的物质从镜子里涌出，化作一个人形。

"你......"

它发出沙哑的声音。

"你怎么知道——"

话没说完。

它就像是失去了力量一样，瘫软在地，化作一滩黑水。

【恭喜，你发现了隐藏规则：与镜中的"自己"同步，可击杀怪谈核心】

【第5次模拟结束·状态：通关+击杀核心】

林默睁开眼。

现实中，才过去了五分钟。

"搞定了。"

他嘴角微扬，走向电梯。

......

外面。

沈清音焦急地看着手表。

凌晨三点整——怪谈激活了。

林默已经进去两分钟了。

"他疯了。"

她咬着嘴唇。

"一个F级觉醒者，挑战C级怪谈？"

"就算有什么特殊能力，也不可能——"

话没说完。

写字楼里传来一声巨响。

然后——

大门打开。

林默走了出来。

身后跟着三个人——正是被困两天的那三个觉醒者。

他们的表情呆滞，像是见了鬼一样。

"你......"

沈清音看着林默，说不出话来。

林默走到她面前，淡淡道：

"搞定了。"

"C级怪谈【凌晨三点的电梯】，核心已击杀，永久消除。"

"现在......"

他看向沈清音身后的陈老。

陈老不知何时已经赶到，同样一脸震惊。

"我的条件，您考虑好了吗？"

陈老沉默了几秒。

然后，他笑了。

"小子，你还真是......让老头子开眼了。"

"从今天起——"

"你就是调查局的特聘顾问。"

"享受A级待遇，行动自由，任务自选。"

"够不够？"

林默笑了笑。

"够了。"

他转身离开。

那三个被救出来的觉醒者这时才反应过来，连忙追上去。

"等等！这位兄弟！"

"你是怎么做到的？"

"两分钟！你两分钟就通关了一个C级怪谈！"

"而且还击杀了核心！"

"这怎么可能！"

林默没有回答。

只是淡淡道：

"没什么。"

"我只是死了五次而已。"

三人："？？？"

他们完全听不懂。

沈清音站在原地，看着林默远去的背影。

她想起了林默说的话——

"我可以在进入怪谈之前，进行'模拟'。"

"在模拟中，我会经历完整的怪谈流程。"

"但模拟中的死亡不是真的死亡——只是一次'试错'。"

"我可以无限模拟，直到找出通关方法。"

无限模拟......

无限试错......

她突然明白了这个能力有多恐怖。

对于普通觉醒者来说，怪谈只有一次机会，死了就是死了。

但对于林默——

他可以无限次地"死"在怪谈里，然后站起来继续尝试。

直到找出正确答案。

"这个人......"

沈清音喃喃道。

"简直就是怪谈的克星。"
