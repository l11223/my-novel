# 第054章 深渊列车

午夜12点。

地铁站深处。

林默站在那扇铁门前。

"咔嚓——"

门锁自动打开了。

铁门缓缓向两侧滑开，露出一条漆黑的隧道。

隧道深处，传来一阵低沉的轰鸣声。

那是列车驶近的声音。

"来了......"

林默深吸一口气。

"开始模拟。"

【第1次模拟开始】

......

模拟中。

林默站在站台上。

一辆老旧的列车从隧道深处驶来。

列车的外观很诡异——车厢是深红色的，窗户里透出暗淡的灯光。

列车停下。

车门打开。

里面......一个人都没有。

"规则1：列车到站后必须上车。"

林默看了眼站台上的规则牌，迈步走上车。

车厢内部比想象的更加阴森。

座椅是黑色的，地板上有暗红色的污渍，空气中弥漫着一股腐朽的气味。

"叮——"

车门关闭。

列车启动。

"欢迎乘坐深渊列车。"

一个机械般的声音响起。

"本次列车共有七节车厢。"

"终点站是——深渊。"

"请遵守车上规则，祝您旅途愉快。"

林默环顾四周。

车厢里贴着更多的规则：

【车厢规则】
```
1. 听到报站后，不要下车
2. 若有人坐到你旁边，不要说话
3. 若看到穿红衣的乘客，假装没看见
4. 若听到婴儿哭声，捂住耳朵
5. 每隔三站，必须换一节车厢
6. 不要进入第七节车厢
7. 若看到窗外有人招手，不要回应
......
```

"和之前的规则有重复......"

林默皱眉。

"但也有新的规则。"

"先验证一下。"

他坐在座位上，等待。

列车继续行驶。

五分钟后——

"第一站到了。请下车的乘客......"

报站声响起。

规则1说听到报站后不要下车。

林默没有动。

车门打开，又关上。

列车继续前进。

十分钟后——

"第二站到了......"

林默依然没有动。

又十分钟——

"第三站到了......"

规则5说每隔三站必须换一节车厢。

林默站起来，向下一节车厢走去。

他推开连接门——

下一节车厢里，坐着一个"人"。

那个"人"穿着西装，面无表情。

但它的脸......没有五官。

林默没有看它，径直走过去，坐到另一排座位上。

规则2：若有人坐到你旁边，不要说话。

那个"人"没有坐到他旁边。

好。

列车继续行驶。

又是三站。

林默再次换车厢。

第三节车厢、第四节车厢、第五节车厢......

每一节车厢里，都有越来越多的"乘客"。

它们形态各异——有的没有脸，有的没有手，有的身体扭曲成不可能的角度。

但只要不违反规则，它们就不会攻击。

林默继续前进。

当他走到第六节车厢门口时——

他停下了。

规则6说不要进入第七节车厢。

第六节车厢是最后可以进入的。

"第七节车厢......是核心所在？"

他看向连接第七节车厢的门。

门是锁着的。

上面贴着一行字：

【终点站到达后，此门将会打开】

"终点站......"

林默皱眉。

"也就是说，必须在列车上待到终点，才能接触核心。"

"那终点站是什么时候？"

"叮——"

列车停下了。

"终点站到了。"

"欢迎来到——深渊。"

车门打开。

林默走向车门，看向外面——

外面是一片......无尽的黑暗。

不是普通的黑暗，而是像"永夜之境"一样的虚无。

"这是......"

他愣住了。

"'永夜'的碎片？"

就在这时——

第七节车厢的门，缓缓打开了。

从里面，走出一个"东西"。

那是一个......列车长。

它穿着老旧的制服，戴着帽子，手里拿着一个检票器。

但它的脸——

是一张扭曲的笑脸。

"乘客您好。"

它的声音沙哑而诡异。

"请出示您的车票。"

林默愣住了。

车票？

他没有车票。

"没有车票的乘客——"

"列车长"的笑容变得更加扭曲。

"将会被'永远'留在这里。"

它向林默走来。

林默转身想跑——

但车门已经关上了。

"列车长"的手抓住了他的肩膀——

死亡。

【第1次模拟结束·死亡原因：没有车票】

"车票......"

林默睁开眼。

"规则里没有提到车票......"

"这是隐藏条件。"

"必须找到车票，才能通过最终考验。"

"车票在哪？"

他开始思考。

"30条规则里，有没有和'车票'相关的线索？"

他回忆着那些规则。

突然——

他想起了一条。

【规则17：若在座位下发现东西，可以拿走】

"座位下......"

他的眼睛亮了。

"车票可能就藏在某个座位下面！"

【第2次模拟开始】
