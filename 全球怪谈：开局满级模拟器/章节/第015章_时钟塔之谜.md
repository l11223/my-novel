# 第015章 时钟塔之谜

【第1次模拟】

林默走进钟楼。

里面比想象的更阴暗。

只有零星的月光从破碎的窗户透进来，照亮斑驳的墙壁。

他站在一楼大厅，环顾四周。

大厅中央是一个旋转楼梯，通向上方。

规则说，到达顶层可以离开。

那么第一步——上楼。

林默迈步走向楼梯。

刚踏上第一级台阶——

"咚——"

钟声响了。

规则4：若听到钟声，必须停止一切动作。

林默立刻停住。

一秒、两秒、三秒......

钟声消失。

他继续前进。

一楼、二楼、三楼......

每爬一层，他都会感到一阵诡异的眩晕。

规则3：每爬一层楼，时间会倒退一小时。

他看了眼手表。

进来的时候是晚上10点。

现在爬到三楼，手表显示的是晚上7点。

"果然......"

他继续往上爬。

四楼、五楼、六楼......

手表的指针继续倒转。

晚上4点、下午1点、上午10点......

到达七楼的时候，他注意到一件事——

他的影子，在动。

规则5：若看到自己的影子在移动，立即逃跑。

林默转身就跑。

但已经来不及了。

影子从地面"站"了起来，化作一个漆黑的人形。

它扑向林默——

死亡。

【第1次模拟结束·死亡原因：被影子杀死】

"影子......从七楼开始出现。"

林默睁开眼，记下这个信息。

"继续。"

【第2次模拟开始】

这一次，他爬到七楼后，立刻转身往下跑。

成功逃脱了影子。

但问题是——往下跑，时间会变回去吗？

他看了眼手表。

从七楼跑到六楼，手表从上午7点变成了上午6点。

"不对......"

他皱眉。

"上楼时间倒退，下楼时间也倒退？"

"那岂不是永远走不完？"

他继续往下跑。

五楼、四楼、三楼......

手表的时间继续倒退。

上午3点、凌晨12点、晚上9点......

当他跑到一楼的时候——

手表显示的是晚上3点。

"等等......"

林默愣住了。

"我进来的时候是晚上10点，爬了6层，时间倒退到上午7点。"

"然后跑下来，又倒退了7小时，变成晚上3点......"

"不对，应该是变成上午12点才对！"

他低头看着手表，陷入沉思。

"时间的变化......不是线性的？"

"还是说——"

"上楼和下楼的规则不一样？"

【第3次模拟开始】

这一次，他仔细记录每一层的时间变化。

```
一楼：22:00
二楼：21:00（-1小时）
三楼：19:00（-2小时）
四楼：16:00（-3小时）
五楼：12:00（-4小时）
六楼：07:00（-5小时）
七楼：01:00（-6小时）
```

"原来如此......"

林默恍然大悟。

"不是每层固定倒退1小时——"

"而是递增的。"

"第一层倒退1小时，第二层倒退2小时，第三层倒退3小时......"

"那么七层加起来是1+2+3+4+5+6=21小时。"

"22:00减去21小时，正好是凌晨1点。"

他继续计算。

"如果继续往上爬......"

"八楼会倒退7小时，手表变成前一天晚上6点。"

"九楼会倒退8小时，变成前一天上午10点。"

"十楼会倒退9小时，变成前一天凌晨1点......"

"而规则7说，在塔内停留超过12小时，将永远困在里面。"

他眯起眼睛。

"这是一个时间陷阱。"

"表面上时间在倒退，但实际上，我在塔内的'真实时间'是在前进的。"

"如果我计算错误，很可能在不知不觉中超过12小时。"

"那么......"

"这个怪谈的核心，就是——"

"让你在时间混乱中迷失，然后困死在塔里。"

【第3次模拟结束·死亡原因：时间超过12小时】

林默睁开眼，嘴角露出笑意。

"找到了。"

"这个怪谈的关键，是'时间计算'。"

"只要算清楚每一层的时间变化，就能找出最优路线。"

"接下来——"

"让我看看，那18条规则里，还有多少是假的。"

【第4次模拟开始】

......

【第50次模拟】

林默靠在墙上，大口喘气。

又死了。

这一次是因为违反了规则11——"若看到镜子中有人影，不要靠近"。

他以为这是假规则，结果......

"规则11是真的。"

他记下。

到目前为止，他已经模拟了50次，找出了以下规律：

【真规则】
- 规则1：只能在整点进入
- 规则3：每层时间倒退（递增）
- 规则4：听到钟声必须停止
- 规则5：影子移动必须逃跑
- 规则7：停留超过12小时永远困住
- 规则9：不能触碰任何时钟
- 规则11：镜子中的人影危险
- 规则14：必须在钟声响起时回答"几点了"
- 规则16：看到穿白色衣服的人必须闭眼

【假规则】
- 规则2：不能回头（假，可以回头）
- 规则6：到达顶层可离开（半真半假，需要满足条件）
- 规则8：必须按顺序爬楼（假）
- 规则10：不能说话（假）

【待定】
- 规则12、13、15、17、18

"还有5条规则没确定......"

林默揉了揉太阳穴。

A级怪谈果然复杂。

50次模拟，还没完全摸清规则。

"继续。"

【第51次模拟开始】
