# 第068章 最后的碎片

从西伯利亚回来后。

林默清点了一下自己的收获。

【永夜碎片收集进度】

1. 【镜中世界】核心 - ✓
2. 【深渊列车】核心 - ✓
3. 【血色医院】核心 - ✓
4. 【无尽楼梯】核心 - ✓
5. 【噩梦领域】核心 - ✓
6. 【死亡森林】核心 - ✓
7. 未知 - ?

"六枚了......"

林默看着手中的六颗黑色宝石。

"还差最后一枚。"

他打开电脑，查看沈清音发来的情报。

【最后一枚碎片可能的位置】

经过全球范围的排查，最后一枚碎片最有可能位于以下地点：

1. **【时间裂缝】**（南美洲·SS级）
   - 可能性：60%
   - 难度：极高

2. **【虚空之门】**（南极·SS级）
   - 可能性：30%
   - 难度：未知

3. **【深海之眼】**（太平洋·SS级）
   - 可能性：10%
   - 难度：极高（需特殊设备）

"三个可能的位置......"

林默思考着。

"【时间裂缝】的可能性最高。"

"但'时间'相关的怪谈......应该是最难攻略的。"

"【虚空之门】位置偏远，情报也不足。"

"【深海之眼】需要深海作业，太麻烦了。"

他权衡利弊。

"先去【时间裂缝】。"

"如果不是的话，再去其他地方。"

就在这时——

办公室的门被敲响了。

"请进。"

门开了。

是周老。

"林默。"

他的表情很严肃。

"有紧急情况。"

"什么情况？"

"【时间裂缝】......爆发了。"

林默心中一震。

"爆发？"

"是的。"

周老走到林默面前。

"三小时前，【时间裂缝】突然开始扩张。"

"它吞噬了周围五十公里的区域。"

"而且——"

他的声音变得低沉。

"扩张还在继续。"

"按目前的速度，一周之内，它会吞噬整个南美洲。"

"一个月之内......整个地球都会被卷入。"

林默站起来。

"我这就去。"

"等等。"

周老拦住他。

"这次不一样。"

"【时间裂缝】的扩张......不是自然发生的。"

"是什么？"

"是......'永夜'。"

林默愣住了。

"'永夜'？它不是被我击败了吗？"

"它确实被击败了。"

周老的表情很凝重。

"但它没有死。"

"你当初只是摧毁了它的核心，把它打回了'暗界'。"

"现在......它在暗界恢复了力量。"

"它通过【时间裂缝】，向这个世界释放力量。"

"想要......重新降临。"

林默沉默了。

他早就知道"永夜"会回来。

只是没想到......这么快。

"我明白了。"

他拿起背包。

"我这就去【时间裂缝】。"

"找到最后一枚碎片。"

"然后——"

他的眼中闪过一丝锐利。

"进入暗界。"

"彻底消灭'永夜'。"

周老看着他。

"你确定？"

"确定。"

林默转身，向门口走去。

"这是我一直以来的目标。"

"消灭'永夜'，结束这场灾难。"

"现在——"

"终于到了最后的时刻。"

他推开门。

"最后一战......"

"开始了。"

......

当天下午。

京都机场。

林默登上了飞往南美洲的航班。

沈清音站在登机口外，目送他离开。

"一定要活着回来......"

她低声说。

飞机起飞，消失在云层中。

林默坐在机舱里，看着窗外的天空。

他的表情平静，心中却在思考。

"'永夜'要复苏了......"

"最后一枚碎片，就在【时间裂缝】里。"

"只要拿到它，我就可以进入暗界。"

"但......"

"暗界是'永夜'的主场。"

"在那里，它几乎是无敌的。"

"我......能赢吗？"

他闭上眼睛。

"不管怎样——"

"我都要去试试。"

"因为......"

"这是我的命运。"
