# 第036章 新的怪谈

一周后。

林默收到了一个新的任务。

"江城郊区出现了一个A级怪谈。"

陈老站在办公桌后，表情严肃。

"调查局已经派人去勘察过了，初步判断规则数量在15-20条之间。"

"是什么类型的怪谈？"

"学校。"

陈老打开文件。

"一所废弃的中学。"

"三天前，有人报警说在那里看到了诡异的现象。"

"我们派人去调查，发现那里已经被怪谈吞噬了。"

"名字呢？"

"我们暂时命名为【永夜学园】。"

林默挑了挑眉。

永夜......

这个名字让他想起了沈清音父亲死亡的那个SS级怪谈——【永夜】。

"有关联吗？"

"不确定。"

陈老摇头。

"但名字的巧合，让我们不得不重视。"

"如果这个A级怪谈和那个SS级怪谈有关......"

"情况可能比我们想象的更复杂。"

林默点点头。

"我接这个任务。"

"你确定？"

"确定。"

林默的眼中闪过一丝锐利。

"如果它真的和【永夜】有关——"

"我更要去。"

陈老看着他，沉默了几秒。

"好。"

"但你要小心。"

"这次任务，我会派沈清音和你一起去。"

"她对【永夜】有执念，让她参与也好。"

"明白。"

......

当天下午。

林默和沈清音来到了那所废弃的中学。

学校的大门锈迹斑斑，铁锁已经断裂。

门柱上的校名已经模糊不清，只能依稀辨认出"江城第三中学"几个字。

"就是这里。"

沈清音看着学校，眼神复杂。

"能感觉到吗？"

"嗯。"

林默点头。

一股诡异的压迫感从学校内部传来。

这是A级怪谈特有的气息。

"规则呢？"

"在门口。"

沈清音指向大门旁边的墙壁。

墙壁上贴着一张泛黄的纸。

上面写着密密麻麻的字：

【永夜学园·入学须知】

```
1. 只能在夜间22:00进入
2. 进入后必须穿校服
3. 男生穿男生校服，女生穿女生校服
4. 听到铃声必须进入最近的教室
5. 上课时间不能离开座位
6. 下课时间不能待在教室里
7. 遇到老师必须问好
8. 遇到校长必须鞠躬
9. 不能进入校长室
10. 不能上楼顶
11. 看到影子在移动，假装没看见
12. 听到有人叫你的名字，不要回应
13. 若看到镜子中的自己在笑，立刻逃跑
14. 午夜12点必须在操场集合
15. 集合时有人点名，必须答"到"
16. 若点名时你的名字出现两次，第二次不要回应
17. 活到早上6点，从正门离开
```

"17条规则......"

林默快速扫了一遍。

"比普通的A级怪谈少一点。"

"但也不能大意。"

沈清音看着那张纸，眼神变得锐利。

"里面有些规则......和【永夜】很像。"

"比如？"

"第12条，'听到有人叫你的名字，不要回应'。"

"还有第16条，'若点名时你的名字出现两次，第二次不要回应'。"

"这两条规则......和我父亲留下的研究资料中的记载一模一样。"

林默沉默了。

"你觉得这个怪谈和【永夜】有关？"

"我不确定。"

沈清音摇头。

"但我的直觉告诉我......这里面有问题。"

"也许——"

"这个怪谈，是【永夜】的......分身？或者子体？"

林默思考着。

如果这个推测是真的......

那么攻略这个A级怪谈，可能是了解SS级怪谈【永夜】的契机。

"我们今晚进去。"

他做出决定。

"先模拟，再行动。"

沈清音点头。

"好。"

......

晚上21:30。

林默和沈清音在学校门口等待。

距离进入时间还有半小时。

林默闭上眼睛。

"开始模拟。"

【第1次模拟开始】

......

半小时后。

林默睁开眼。

"怎么样？"

沈清音问。

"模拟了15次。"

林默的表情很平静。

"已经大致摸清了规则的真假。"

"17条规则中，3条是假的，2条是陷阱。"

"真正需要遵守的，有12条。"

"那么快？"

沈清音有些惊讶。

"A级怪谈的模拟，至少需要上百次才能完全摸清......"

"我用了【多线模拟】。"

林默解释。

"Lv.4的新能力，可以同时进行两条模拟线路。"

"效率翻倍。"

沈清音恍然大悟。

"原来如此......"

"时间到了。"

林默看了眼手表。

22:00。

"走吧。"

他推开学校大门，走了进去。

沈清音跟在后面。

两人的身影消失在黑暗中。

A级怪谈【永夜学园】——

攻略开始。
