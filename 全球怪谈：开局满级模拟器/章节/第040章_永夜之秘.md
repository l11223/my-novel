# 第040章 永夜之秘

沈清音出院后。

林默开始研究他在【永夜学园】获得的特殊信息。

【永夜之秘】

```
永夜，是一切怪谈的源头。

三年前，它从"另一个世界"降临。

它的本体是一片"黑暗"——无边无际的黑暗。

它吞噬了无数生命，然后将它们转化为"怪谈"。

每一个怪谈，都是它的"碎片"。

碎片越多，它就越强。

当碎片数量达到一定程度——

永夜将彻底降临这个世界。

届时，整个世界都将陷入"永夜"。

再无白昼。

【永夜本体信息】

等级：超越S级（暂定SSS级）

位置：未知

核心：未知

弱点：未知

【相关信息】

沈建国（沈清音之父）曾接触过永夜的"一缕意识"。

他在临死前留下了一份研究报告。

报告中记录了永夜的部分特征和可能的弱点。

报告存放位置：京都·怪谈研究所·绝密档案室
```

"SSS级......"

林默看着这些信息，眉头紧锁。

他原本以为SS级就是最强的怪谈了。

没想到，还有超越SS级的存在。

"如果'永夜'是所有怪谈的源头......"

"那我之前消灭的那些怪谈，只不过是它的'碎片'而已。"

"真正的本体......比那些碎片强大无数倍。"

他陷入沉思。

以他现在的实力，别说SSS级，就算是SS级怪谈，他也没有把握能攻略。

"需要变得更强。"

"但怎么变强呢？"

他看向模拟器面板。

【满级模拟器·状态面板】

**宿主**：林默

**等级**：Lv.4

**已通关怪谈**：22（D级×15，C级×1，B级×1，A级×3，S级×1）

**击杀核心**：6

**能力**：
- 【模拟】
- 【记忆保留】
- 【加速模拟】（2倍速）
- 【规则标记】
- 【多线模拟】

**异能**：【分析】C级

**资产**：约1500万

"Lv.4......还不够。"

林默思考着。

"要挑战更高级别的怪谈，就需要更强的模拟能力。"

"Lv.5会解锁什么能力呢？"

"也许是更快的模拟速度？或者更多的模拟线路？"

他站起来，走到窗前。

"不管怎样——"

"下一步的目标明确了。"

"第一，找到沈建国的研究报告。"

"第二，继续攻略怪谈，提升模拟器等级。"

"第三，寻找'永夜'本体的位置。"

"第四——"

他的眼中闪过一丝决然。

"消灭它。"

......

当天下午。

林默去找陈老。

"你想要沈建国的研究报告？"

陈老看着林默，表情复杂。

"是的。"

"为什么？"

"因为我要消灭'永夜'。"

陈老沉默了。

"你知道那是什么等级的怪谈吗？"

"超越S级。"

林默点头。

"可能是SSS级。"

"你还要去？"

"必须去。"

林默看着陈老的眼睛。

"'永夜'是所有怪谈的源头。"

"只有消灭它，才能彻底结束这场灾难。"

"否则，怪谈会无穷无尽地出现，人类永远无法安宁。"

陈老叹了口气。

"你说得对。"

"但那份报告......不在我这里。"

"在哪？"

"京都。"

陈老站起来。

"怪谈研究所的绝密档案室。"

"只有总局高层才有权限调阅。"

"我只是江城分局的局长，没有那个权限。"

林默皱眉。

"那怎么办？"

"只有两个办法。"

陈老伸出两根手指。

"第一，你加入总局，成为正式成员。然后慢慢升职，获得权限。"

"第二——"

他看着林默。

"你做出足够大的贡献，让总局主动把报告给你。"

"足够大的贡献......"

林默思考着。

"比如？"

"比如——"

陈老嘴角微微上扬。

"消灭一个SS级怪谈。"

林默愣了一下。

"SS级？"

"对。"

陈老点头。

"如果你能独自消灭一个SS级怪谈的核心——"

"不管是总局还是研究所，都会对你另眼相看。"

"到时候，别说一份研究报告——"

"就算是更重要的东西，他们也会主动送上门来。"

林默沉默了。

SS级怪谈......

全球只有十个左右。

每一个都是国家级的灾难。

他现在的实力......真的能挑战吗？

"我知道你在想什么。"

陈老看着他。

"你觉得自己还不够强。"

"但你忘了一件事。"

"什么？"

"你的能力......是独一无二的。"

陈老的眼中闪过一丝光芒。

"只要给你足够的时间——"

"你可以攻略任何怪谈。"

"SS级......也不例外。"

林默看着陈老，沉默了很久。

然后——

他笑了。

"您说得对。"

"我差点忘了。"

他转身，向门口走去。

"我去准备。"

"SS级怪谈——"

"我来了。"
