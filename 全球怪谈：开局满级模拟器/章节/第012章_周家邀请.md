# 第012章 周家邀请

三天后。

林默收到了正式的邀请函。

【周家诚邀林默先生于本周六晚出席家宴，共商合作事宜。】

落款是周家家主——周德海。

"家宴？"

林默看着邀请函，若有所思。

表面上是请吃饭，实际上恐怕没那么简单。

周家是江城最大的觉醒者家族，底蕴深厚，势力庞大。

这种家族邀请他去——不可能只是为了"感谢救命之恩"。

"去还是不去？"

沈清音站在一旁，问道。

"去。"

林默把邀请函放进口袋。

"为什么？"

"因为他们有我想要的东西。"

"什么东西？"

"资源。"

林默抬起头。

"A级怪谈的资料、特殊道具、高级药剂......"

"这些东西，调查局虽然有，但申请流程太麻烦。"

"周家作为本地最大的觉醒者家族，肯定有自己的渠道。"

"如果能和他们建立关系......"

"我以后的发展会顺利很多。"

沈清音看着他，沉默了几秒。

"你想得很远。"

"不远。"林默摇头，"只是务实而已。"

"想要变强，光靠自己不够。"

"需要借势。"

沈清音点点头。

"那我陪你去？"

"不用。"

"为什么？"

"我一个人足够了。"

林默看了她一眼。

"而且......你是调查局的人。"

"我和周家谈合作，你在场不太方便。"

沈清音愣了一下。

"你的意思是......你想绕开调查局？"

"不是绕开，是......多一条路。"

林默的表情很平静。

"调查局是官方，周家是民间。"

"两边的资源不冲突。"

"我两边都要。"

沈清音看着他，突然笑了。

"你这人......还真是贪心。"

"贪心才能活得久。"

......

周六晚上。

周家别墅。

这是一栋占地极大的庄园，位于江城郊区最繁华的地段。

光是门口的警卫，就有十几个——全都是觉醒者。

林默走到门口，出示邀请函。

"林默先生，请跟我来。"

一个穿着旗袍的女管家迎了上来。

"老爷和少爷已经等候多时了。"

林默跟着她穿过长长的走廊，来到一间宴会厅。

厅里只坐着两个人。

一个是头发花白的老人——周老。

另一个是周天昊。

"林兄！你来了！"

周天昊跳起来，热情地迎上来。

"快快快，坐这边！"

林默点点头，在他旁边坐下。

"林先生。"

周老微微一笑。

"久仰大名。"

"周老客气了。"

"不客气。"周老摆摆手，"你救了我孙子的命，这份情，我周家记着。"

"今天请你来，一是为了感谢，二是......"

他顿了顿。

"想和你谈一笔生意。"

"什么生意？"

"一个A级怪谈。"

周老的表情变得严肃。

"三个月前，我们周家在老城区发现了一个A级怪谈。"

"本来想报告给调查局，让他们处理。"

"但后来我们发现......这个怪谈里，有我们周家的东西。"

"什么东西？"

"一件传家宝。"

周老的眼中闪过一丝复杂。

"具体是什么，我不方便透露。"

"但我可以告诉你——"

"那件东西对周家非常重要，绝对不能落入外人之手。"

"所以，你们想自己解决？"

"对。"

周老点头。

"调查局处理怪谈，会把里面所有的东西都收走。"

"我们不能接受这个结果。"

"所以才找到你。"

林默沉默了几秒。

"你们想让我......帮你们攻略这个A级怪谈？"

"对。"

"条件呢？"

"怪谈里的其他东西，全部归你。"

"我们只要那件传家宝。"

"另外，报酬是200万现金，外加周家的资源支持。"

"什么资源？"

"任何你需要的东西，只要周家有，都可以提供。"

林默挑了挑眉。

这个条件......相当优厚。

200万现金，加上周家的资源支持，再加上怪谈里的战利品——

怎么算都是赚的。

"怪谈的具体信息呢？"

"我已经准备好了。"

周老拍了拍手。

那个女管家走进来，递上一个文件夹。

林默接过，翻开。

【A级怪谈：老城区·时钟塔】

【位置】：江城老城区废弃钟楼

【发现时间】：3个月前

【已知规则】：
```
1. 只能在整点时分进入
2. 进入后不得回头
3. 每爬一层楼，时间会倒退一小时
4. 若听到钟声，必须停止一切动作
5. 若看到自己的影子在移动，立即逃跑
6. 到达顶层可离开
7. 在塔内停留超过12小时，将永远困在里面
......
（规则过长，共计18条）
```

【已尝试次数】：7次

【伤亡人数】：4人死亡，2人重伤，1人失踪

林默仔细看完资料，合上文件夹。

"18条规则，4人死亡，1人失踪......"

他抬起头，看着周老。

"这个怪谈，不简单。"

"我知道。"

周老点头。

"所以我们才找你。"

"普通的觉醒者，根本没办法攻略。"

"但你......不一样。"

他看着林默的眼睛。

"我听天昊说了你的能力。"

"如果你真的能'模拟'——"

"这个怪谈，或许有希望。"

林默沉默了几秒。

然后，他笑了。

"成交。"

周老松了口气。

"太好了。"

"不过——"

林默话锋一转。

"我有一个条件。"

"什么条件？"

"我需要一周的准备时间。"

"这一周内，我要调查这个怪谈的所有信息。"

"你们之前的尝试记录、幸存者的口述、任何相关的资料——"

"我全部都要。"

周老毫不犹豫地点头。

"没问题。"

"一周后——"

林默站起身。

"我会给你们一个满意的答案。"
