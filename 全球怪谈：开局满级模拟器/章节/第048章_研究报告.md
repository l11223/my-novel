# 第048章 研究报告

一周后。

京都，怪谈研究所。

林默坐在档案室里，面前摆着一叠厚厚的资料。

那是沈建国留下的研究报告——关于【永夜】的全部研究成果。

林默一页一页地翻看。

【永夜研究报告·第1号】

日期：三年前

报告人：沈建国

```
今天，我们首次接触到了"永夜"的痕迹。

它不是普通的怪谈。

它是......所有怪谈的源头。

从它的"身体"里，分裂出无数"碎片"。

这些碎片散落在世界各地，形成了各种等级的怪谈。

D级、C级、B级、A级、S级、SS级......

它们都是"永夜"的一部分。
```

【永夜研究报告·第7号】

```
我们发现了"永夜"的一个重要特征——

它没有固定的形态。

它是一片"黑暗"。

无边无际的黑暗。

这片黑暗可以吞噬一切，包括光、物质、甚至......时间和空间。

在它的"领域"内，一切规则都由它制定。

没有人能违抗它的规则。
```

【永夜研究报告·第15号】

```
今天，我终于找到了"永夜"的一个可能的弱点。

它虽然是一片黑暗，但黑暗的中心......有一个"核心"。

那个核心，是"永夜"力量的源泉。

如果能摧毁那个核心......

"永夜"就会消失。

所有的怪谈，也会随之消失。

但问题是——

没有人知道核心在哪里。

也没有人知道如何摧毁它。
```

【永夜研究报告·第23号】

```
我找到了一些线索。

"永夜"的核心，不在这个世界。

它在......另一个"维度"。

那个维度，被称为"永夜之境"。

只有进入那个维度，才能接触到核心。

而进入那个维度的方法......

需要一把"钥匙"。

那把钥匙，就藏在某个SS级怪谈里。
```

【永夜研究报告·第31号】（最后一份报告）

```
我找到了"钥匙"的位置。

它在......【镜中世界】。

【镜中世界】的核心，就是通往"永夜之境"的钥匙。

只要摧毁那个核心......

就能打开通往"永夜"的大门。

但我没有时间了。

"永夜"发现了我的研究。

它......在追杀我。

如果这份报告被发现......

希望后来人能继承我的研究。

找到"永夜"。

消灭它。

结束这场噩梦。

——沈建国 绝笔
```

林默合上报告，陷入沉思。

"【镜中世界】的核心，是通往永夜之境的钥匙......"

"但我已经摧毁了那个核心......"

"这意味着什么？"

他思考着。

"要么，钥匙随着核心一起消失了。"

"要么——"

"钥匙在核心被摧毁时，被释放了出来。"

他回忆起摧毁核心的那一刻。

镜球碎裂的时候......

确实有一道奇怪的光芒，射向了某个方向。

当时他没有注意。

但现在想来......

"那道光......会不会就是钥匙？"

"它飞向了......哪里？"

就在这时——

他的手机响了。

是沈清音。

"林默，出事了！"

她的声音很急促。

"什么事？"

"江城......出现了异常！"

"一道黑暗......正在吞噬城市！"

林默的瞳孔骤缩。

"黑暗？"

"是的！"

沈清音的声音在颤抖。

"陈局长说......可能是【永夜】！"

"【永夜】现身了！"

林默握紧手机。

"我马上回去。"
