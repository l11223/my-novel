# 📋 网文创作诊断清单

> 作品出问题时，用这个清单快速定位并修复

---

## 🔴 紧急问题诊断

### 数据差（追读低/收藏少）

**症状**：读者看完不追更，收藏少

**可能原因 & 解决方案**：

| 检查项 | 是否命中 | 解决方案 |
|--------|----------|----------|
| 开头太慢，3章内没爽点 | □ | 重写开头，第1章就上冲突 |
| 主角太弱太久 | □ | 加速成长，给更多装逼机会 |
| 金手指存在感低 | □ | 增加金手指出场，展示威力 |
| 剧情太平淡 | □ | 注入危机，用"危机注入器" |
| 没有钩子，章末没悬念 | □ | 每章结尾加钩子 |
| 更新不稳定 | □ | 保持日更，稳定时间 |

---

### 评论区骂声一片

**症状**：读者愤怒，要弃书

**可能原因 & 解决方案**：

| 检查项 | 是否命中 | 解决方案 |
|--------|----------|----------|
| 主角圣母，对敌人心软 | □ | 让主角果断，该杀就杀 |
| 女主跟别人暧昧 | □ | 立刻澄清/解决，不要拖 |
| 主角被虐太惨没翻身 | □ | 安排爽点，让主角反击 |
| 反派太蠢/太弱 | □ | 给反派加智商，增强威胁 |
| 配角抢戏 | □ | 减少配角戏份，聚焦主角 |
| 节奏太慢 | □ | 删水字，加快节奏 |

---

### 写不下去了（卡文）

**症状**：不知道接下来写什么

**快速解决方案**：

```
1. 用"副本生成器"随机生成下一个任务
2. 用"危机注入器"制造意外事件
3. 让一个旧敌人/旧配角重新出现
4. 时间跳跃："三天后——"直接推进
5. 引入新角色打破僵局
6. 给主角一个紧急任务
7. 让主角发现一个秘密
8. 突然来一场追杀
```

## 🟡 常见问题诊断

### 问题1：战力崩了

**症状**：前面能打赢元婴，后面打不过金丹

**诊断步骤**：
1. 回看之前的战斗章节
2. 列出主角每个阶段的战力
3. 找到矛盾点

**修复方案**：
- 补充解释（受伤/环境限制/敌人用了秘法）
- 调整后面的设定
- 如果实在改不了，当作"伏笔"后面解释

---

### 问题2：人设漂移

**症状**：主角/配角性格变了，读者觉得OOC

**诊断步骤**：
1. 回看角色档案，确认原设定
2. 找到性格变化的章节
3. 判断是"成长"还是"漂移"

**修复方案**：
- 如果是成长，补充铺垫和心理变化
- 如果是漂移，修改让性格回归
- 添加过渡，让变化合理化

---

### 问题3：配角失踪

**症状**：重要配角消失了，读者追问

**诊断步骤**：
1. 检查配角追踪表
2. 确认配角最后出场时间
3. 规划后续戏份

**修复方案**：
- 安排配角重新出场
- 如果不好写，用旁白/对话交代近况
- 设定中说明配角去了哪里

---

### 问题4：伏笔忘了

**症状**：埋了伏笔忘了回收

**诊断步骤**：
1. 检查伏笔清单
2. 找到未回收的伏笔
3. 规划回收时机

**修复方案**：
- 尽快安排回收
- 如果已经来不及，当作"长线伏笔"
- 实在没法回收，模糊处理

---

### 问题5：节奏失控

**症状**：连续几章都很无聊/都在打架

**诊断步骤**：
1. 统计最近10章的类型
2. 检查节奏比例是否合理

**理想比例**：
- 战斗章：25-35%
- 剧情推进章：35-45%
- 日常/感情章：15-25%
- 高潮章：10-15%

**修复方案**：
- 太多战斗→加日常/感情戏
- 太多日常→加冲突/危机
- 太平淡→加爽点

## 🟢 预防性检查

### 每章必检（写完后）

- [ ] 有爽点吗？
- [ ] 有钩子吗？
- [ ] 推进剧情了吗？
- [ ] 对话占比够吗（>40%）？
- [ ] 有AI味吗？
- [ ] 战力一致吗？
- [ ] 人设一致吗？

### 每10章必检

- [ ] 主线进度合理吗？
- [ ] 伏笔都记录了吗？
- [ ] 角色状态都更新了吗？
- [ ] 节奏比例合理吗？
- [ ] 有没有配角失踪？

### 每卷必检

- [ ] 本卷有大高潮吗？
- [ ] 本卷伏笔都埋了吗？
- [ ] 本卷BOSS死了吗？
- [ ] 下卷钩子留了吗？
- [ ] 角色都有结局/交代吗？

## 🔧 快速修复工具箱

### 提升爽感
```
→ 加一场打脸戏
→ 让主角获得宝物
→ 让主角突破境界
→ 让美女对主角表白
→ 让敌人跪下求饶
```

### 增加紧张感
```
→ 给主角设置时限
→ 让主角身份暴露
→ 有人绑架主角在乎的人
→ 突然出现强敌
→ 金手指突然出问题
```

### 推动剧情
```
→ 新角色登场
→ 收到紧急任务
→ 发现重要线索
→ 旧敌人重新出现
→ 时间跳跃直接推进
```

### 制造反转
```
→ 队友背叛
→ 身份揭示
→ 真相曝光
→ 绝境逆袭
→ 隐藏实力曝光
```
